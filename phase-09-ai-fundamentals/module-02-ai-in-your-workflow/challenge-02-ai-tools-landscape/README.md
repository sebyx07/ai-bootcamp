# Challenge: AI Tools Landscape

## What you'll learn
An overview of the major AI tools beyond Claude Code — what exists, what each tool is best at, and how to choose the right tool for different tasks.

## Watch first
- [Large Language Models explained briefly — 3Blue1Brown](https://www.youtube.com/watch?v=LPZh9BOjkQs) (12 min) — Understand the underlying technology that powers all these tools

> "I'm ready for the challenge"

## You're done when
- [ ] You can name and describe at least five major AI tools
- [ ] You understand the difference between chat-based AI, coding assistants, and AI-enabled editors
- [ ] You know that different tools have different strengths
- [ ] You have a sense of which tool you would reach for in different situations

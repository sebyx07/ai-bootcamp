# Phase 08: AI Fundamentals

You have been using AI throughout this bootcamp. Now you'll understand how it actually works.

This phase is mostly conceptual. There is minimal coding. Instead, you'll have discussions with Claude about how large language models work, why they sometimes get things wrong, and how to think critically about AI tools. You'll come away with a mental model that makes you a smarter, more effective AI user.

## Modules

1. **How AI Works** — What an LLM is, tokens and context windows, why AI makes mistakes
2. **AI in Your Workflow** — When to use AI, the AI tools landscape, limitations and ethics

## Prerequisites

- Completed Phases 01-07 (or equivalent experience using AI coding tools)

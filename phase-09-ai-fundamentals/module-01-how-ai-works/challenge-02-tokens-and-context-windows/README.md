# Challenge: Tokens and Context Windows

## What you'll learn
What tokens are, what a context window is, and why these concepts directly affect how you use AI tools.

## Watch first
- [Large Language Models explained briefly — 3Blue1Brown](https://www.youtube.com/watch?v=LPZh9BOjkQs) (12 min) — Covers how LLMs process text as tokens, with clear visuals
- [Intro to Large Language Models — Andrej Karpathy](https://www.youtube.com/watch?v=zjkBMFhNj_g) (60 min) — Deeper explanation of tokens, context, and how models process information

- [ ] You can explain what a token is with an example
- [ ] You understand what a context window is and why it has a limit

# Challenge: What Is an LLM?

## What you'll learn
What a large language model actually is, how it generates text, and why it is different from a search engine or a database.

## Watch first
- [Large Language Models explained briefly — 3Blue1Brown](https://www.youtube.com/watch?v=LPZh9BOjkQs) (12 min) — Beautiful visual explanation of how LLMs work, from the creator of the best math explainer channel on YouTube
- [Intro to Large Language Models — Andrej Karpathy](https://www.youtube.com/watch?v=zjkBMFhNj_g) (60 min) — Deep but accessible talk by a former OpenAI/Tesla AI lead covering how LLMs are trained and what they can do

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what an LLM is in your own words, without jargon
- [ ] You understand that LLMs predict the next word based on patterns
- [ ] You know the difference between an LLM and a search engine
- [ ] You can explain what "training" means for an AI model

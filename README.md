# AI Bootcamp

A hands-on bootcamp where you learn to build software using AI as your mentor and partner. No prior tech experience needed.

## How this works

1. **You have Claude Code and VS Code ready** — that's all you need
2. **Each phase** covers a big topic (terminal, frontend, backend, etc.)
3. **Each module** breaks that topic into smaller areas
4. **Each challenge** is a hands-on problem you solve with Claude

## Your daily workflow

1. Say **"I want to continue"** to Claude
2. Claude gives you a short overview of the topic
3. Claude tells you to watch a YouTube video (5-25 min)
4. You watch it, come back, say **"ready"**
5. You practice together — Claude writes the code, you direct and learn
6. Claude quizzes you, checks you off, points you to the next challenge
7. Commit your work from `workspace/` to GitHub

## The phases

| Phase | What you'll learn | Days |
|-------|-------------------|------|
| **01 — Foundations** | Claude Code, Terminal, Git, GitHub | 1-3 |
| **02 — Dev Environments** | Package managers, nvm, rbenv, Docker, databases setup | 3-4 |
| **03 — Frontend** | HTML, CSS, SCSS, JavaScript, React | 5-8 |
| **04 — Backend** | How the web works, Node, Ruby, Rails, databases, auth, Redis, S3 | 9-13 |
| **05 — Testing & Quality** | Unit tests, linting, CI/CD | 14 |
| **06 — Infrastructure** | Nginx, deployment, domains, SSL | 15 |
| **07 — DevOps** | Remote servers, SSH, libvirt VMs, systemd, monitoring | 16-17 |
| **08 — Advanced Claude Code** | Prompting, CLAUDE.md, skills, subagents, MCP | 18-19 |
| **09 — AI Fundamentals** | How LLMs work, AI tools, limitations | 19-20 |
| **10 — Capstone** | Build and ship a real project | 21-25 |

## Rules

- **Don't skip the videos** — they give you the theory you need
- **Don't let Claude do everything** — think first, then prompt
- **Short prompts win** — say `check port 3000` not a paragraph
- **Commit often** — push your work to GitHub after each challenge
- **It's okay to be stuck** — that's where learning happens
- **Ask Claude to explain** — if you don't understand something, ask "why?"

## Getting started

1. **Fork this repo** — click the **Fork** button at the top right of this page to create your own copy
2. **Clone your fork:**

```bash
git clone git@github.com:YOUR_USERNAME/ai-bootcamp.git
cd ai-bootcamp
claude
```

3. Then just say: **"I want to continue"** — Claude will find your first challenge and take it from there.

Let's go!

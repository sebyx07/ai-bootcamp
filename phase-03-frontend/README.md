# Phase 03: Frontend

## What you'll learn

How to build what users actually see and interact with. You'll go from writing your first line of HTML to building a complete React dashboard -- all from scratch.

## Modules

| Module | Topic | Challenges | Time estimate |
|--------|-------|------------|---------------|
| **01** | HTML | 6 challenges | 1 day |
| **02** | CSS & SCSS | 8 challenges | 1.5 days |
| **03** | JavaScript | 10 challenges | 2 days |
| **04** | React | 10 challenges | 2 days |

## Prerequisites

- You have completed Phase 01 (Foundations) and Phase 02 (Dev Environments)
- You can use the terminal and Git comfortably
- You have a code editor (VS Code) and Claude Code installed

## How to work through this phase

1. Go module by module, challenge by challenge -- in order
2. Each challenge has a README with a short video to watch first
3. After watching, open Claude Code in the challenge folder and say: **"I'm ready for the challenge"**
4. Claude will teach you, quiz you, and guide you through building something real
5. Commit your work after each challenge before moving on

## What you'll build

By the end of this phase you will have built:
- A multi-page HTML website with forms and semantic markup
- A styled, responsive CSS layout (with SCSS)
- Interactive JavaScript apps that fetch data from APIs
- A React todo app and a data dashboard

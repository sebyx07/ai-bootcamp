# Challenge: Make This Ugly Page Nice

## What you'll learn
How to apply everything you know about CSS to redesign a functional but visually terrible page.

## Watch first
- [CSS Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=yfoY53QXEnI) (85 min) — Review any sections you need; this challenge uses everything from the CSS module

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Claude has given you an ugly (but working) HTML page
- [ ] You restyled it using your own CSS — Claude only helped with hints
- [ ] The page uses good typography, colors, spacing, and layout
- [ ] The page is responsive (looks good on mobile and desktop)
- [ ] You are genuinely proud of how the page looks

# Challenge: The Box Model

## What you'll learn
How every HTML element is a box with content, padding, border, and margin -- and how to control the size of that box.

## Watch first
- [CSS Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=yfoY53QXEnI) (85 min) — Focus on the box model section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can draw and label the 4 parts of the box model (content, padding, border, margin)
- [ ] You have used `box-sizing: border-box` and can explain why it helps
- [ ] You have created elements with visible borders, padding, and margin
- [ ] You used browser DevTools to inspect the box model of an element

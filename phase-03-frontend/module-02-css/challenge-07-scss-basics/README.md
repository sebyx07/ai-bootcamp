# Challenge: SCSS Basics

## What you'll learn
How to use SCSS (Sass) to write CSS more efficiently with variables, nesting, and mixins.

## Watch first
- [Sass Crash Course](https://www.youtube.com/watch?v=nu5mdN2JIwM) (60 min) — Traversy Media covers SCSS variables, nesting, partials, mixins, and more

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a `.scss` file that compiles to CSS
- [ ] You use at least 3 SCSS variables (e.g., colors, font sizes)
- [ ] You use nesting to write cleaner selectors
- [ ] You have created at least 1 mixin and used it
- [ ] You can explain why SCSS needs to be compiled before browsers can use it

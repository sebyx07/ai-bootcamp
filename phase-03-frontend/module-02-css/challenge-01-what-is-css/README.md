# Challenge: What is CSS?

## What you'll learn
How to add CSS to an HTML page and change the look of elements for the first time.

## Watch first
- [CSS Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=yfoY53QXEnI) (85 min) — Watch the first 20 minutes covering CSS basics, selectors, and how to add CSS to HTML

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can add CSS using all 3 methods: inline, internal (`<style>` tag), and external file
- [ ] You have an external `style.css` file linked to your HTML
- [ ] You have changed the color, font-size, and background of elements using CSS
- [ ] You can explain what a CSS selector, property, and value are

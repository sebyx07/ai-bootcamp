# Challenge: Flexbox

## What you'll learn
How to use CSS Flexbox to lay out elements in rows and columns with alignment and spacing control.

## Watch first
- [Flexbox Crash Course](https://www.youtube.com/watch?v=3YW65K6LcIA) (47 min) — Traversy Media covers all Flexbox properties with practical examples
- [Learn Flexbox the Easy Way](https://youtu.be/u044iM9xsWU) (34 min) — Kevin Powell breaks down flexbox concepts step by step

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have built a navigation bar using flexbox
- [ ] You have created a card layout with 3 cards in a row
- [ ] You can center an element both horizontally and vertically using flexbox
- [ ] You can explain `justify-content` vs `align-items`

# Challenge: Grid

## What you'll learn
How to use CSS Grid to build two-dimensional layouts with rows and columns.

## Watch first
- [CSS Grid Crash Course](https://www.youtube.com/watch?v=0xMQfnTU6oo) (28 min) — Traversy Media covers Grid fundamentals with practical layout examples
- [Learn CSS Grid the Easy Way](https://www.youtube.com/watch?v=rg7Fvvl3taU) (37 min) — Kevin Powell walks through Grid step by step with clear examples

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a page layout with a header, sidebar, main content, and footer using Grid
- [ ] You have used `grid-template-columns` and `grid-template-rows`
- [ ] You have used `grid-template-areas` for at least one layout
- [ ] You can explain when to use Grid vs Flexbox

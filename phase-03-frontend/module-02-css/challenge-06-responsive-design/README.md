# Challenge: Responsive Design

## What you'll learn
How to make your web pages look good on any screen size using media queries and responsive techniques.

## Watch first
- [Responsive Design Made Easy](https://www.youtube.com/watch?v=bn-DQCifeQQ) (43 min) — Kevin Powell covers practical responsive design techniques from scratch
- [Build a Responsive Website](https://www.youtube.com/watch?v=p0bGHP-PXD4) (50 min) — Traversy Media builds a responsive layout step by step

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your page looks good on both mobile (375px wide) and desktop (1200px wide)
- [ ] You have used at least 2 media queries with `@media`
- [ ] Your navigation collapses or stacks on small screens
- [ ] You have used relative units (%, em, rem, vw) instead of only px
- [ ] You tested by resizing your browser window

# Challenge: Colors, Fonts, and Spacing

## What you'll learn
How to control the visual appearance of text with colors, font families, font sizes, and spacing properties.

## Watch first
- [CSS Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=yfoY53QXEnI) (85 min) — Focus on the typography and color sections (around 20-40 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your page uses at least 3 different colors (hex, rgb, or named)
- [ ] You have imported and used a Google Font
- [ ] You have adjusted font-size, font-weight, and line-height
- [ ] You have used margin and padding to add spacing
- [ ] Your page looks visually pleasant — not just default browser styles

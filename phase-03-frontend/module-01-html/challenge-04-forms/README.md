# Challenge: Forms

## What you'll learn
How to create HTML forms that collect user input like text, emails, checkboxes, and dropdowns.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — Focus on the forms section near the end of the video

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a form with at least 5 different input types
- [ ] Your form has labels properly connected to each input
- [ ] You have a submit button
- [ ] You can explain what the `name` attribute does on an input
- [ ] You have used at least one `<select>` dropdown or `<textarea>`

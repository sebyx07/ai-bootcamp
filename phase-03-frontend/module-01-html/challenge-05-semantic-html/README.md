# Challenge: Semantic HTML

## What you'll learn
How to use meaningful HTML tags like `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>` instead of generic `<div>` tags everywhere.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — The semantic HTML section near the end covers all the key tags

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your page uses `<header>`, `<nav>`, `<main>`, and `<footer>`
- [ ] You have at least one `<section>` or `<article>`
- [ ] You are NOT using `<div>` where a semantic tag would be better
- [ ] You can explain why semantic HTML matters for accessibility and SEO

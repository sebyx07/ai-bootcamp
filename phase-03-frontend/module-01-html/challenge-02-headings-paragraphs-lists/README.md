# Challenge: Headings, Paragraphs, and Lists

## What you'll learn
How to structure text content using headings, paragraphs, and ordered/unordered lists.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — Focus on the sections about text elements and lists (around 10-25 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your page has headings of at least 3 different levels (h1, h2, h3)
- [ ] You have multiple paragraphs of text
- [ ] You have at least one ordered list and one unordered list
- [ ] You can explain when to use `<ol>` vs `<ul>`

# Challenge: Your First Webpage

## What you'll learn
How to create an HTML file from scratch and view it in your browser.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — Traversy Media walks through every core HTML concept; watch at least the first 15 minutes

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have an `index.html` file in this folder
- [ ] It has a proper HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] It displays a heading and a paragraph in your browser
- [ ] You can explain what each part of the HTML boilerplate does

# Challenge: Fix This Broken Page

## What you'll learn
How to spot and fix common HTML errors by debugging a broken web page.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — Review any sections you found tricky; this challenge tests everything you've learned

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Claude has given you a broken HTML file
- [ ] You identified and fixed all the bugs yourself (Claude helped only with hints)
- [ ] The page displays correctly in the browser
- [ ] You can list at least 3 mistakes that were in the original file

# Challenge: Links and Images

## What you'll learn
How to add links to other pages and images to your HTML page.

## Watch first
- [HTML Crash Course For Absolute Beginners](https://www.youtube.com/watch?v=UB1O30fR-EE) (60 min) — Focus on the anchor tag and image sections (around 25-35 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your page has at least one link to an external website
- [ ] Your page has a link to a second HTML page you created
- [ ] You have at least one image displayed on the page
- [ ] Every image has an `alt` attribute with a description
- [ ] You can explain what the `href` and `src` attributes do

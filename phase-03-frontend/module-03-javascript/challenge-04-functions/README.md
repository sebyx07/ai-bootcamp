# Challenge: Functions

## What you'll learn
How to write reusable blocks of code with functions, parameters, and return values.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Focus on the functions section (around 40-55 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write a function declaration and a function expression
- [ ] You have written an arrow function
- [ ] Your functions accept parameters and return values
- [ ] You have built at least 3 useful functions (e.g., calculate tip, format name, check age)
- [ ] You can explain why functions make code better

# Challenge: Events Everywhere

## What you'll learn
How to respond to all kinds of user interactions: typing, hovering, scrolling, submitting forms, and pressing keys.

## Watch first
- [JavaScript DOM Crash Course - Part 1](https://www.youtube.com/watch?v=0ik6X4DJKCc) (40 min) — Review the events section; builds on what you learned in the previous challenge

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You handle at least 4 different event types (click, input, keydown, submit, mouseover, etc.)
- [ ] You have a text input that shows a live preview as the user types
- [ ] You have a form that prevents default submission and handles data with JavaScript
- [ ] You opened the page in a browser and tested all interactions

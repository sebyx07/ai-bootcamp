# Challenge: DOM — Click a Button

## What you'll learn
How to use JavaScript to interact with HTML elements on a web page — selecting elements and making a button do something when clicked.

## Watch first
- [JavaScript DOM Crash Course - Part 1](https://www.youtube.com/watch?v=0ik6X4DJKCc) (40 min) — Traversy Media covers selecting elements, changing content, and handling events

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can select an HTML element using `document.querySelector()`
- [ ] You have a button that changes text on the page when clicked
- [ ] You have a button that changes the style of an element when clicked
- [ ] You opened the HTML file in a browser and everything works
- [ ] You can explain what the DOM is in your own words

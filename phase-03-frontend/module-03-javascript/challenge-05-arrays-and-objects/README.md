# Challenge: Arrays and Objects

## What you'll learn
How to work with collections of data using arrays (ordered lists) and objects (key-value pairs).

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Focus on arrays and objects sections (around 55-75 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can create arrays and access items by index
- [ ] You have used `push`, `pop`, `map`, `filter`, and `forEach`
- [ ] You can create objects with properties and methods
- [ ] You have used an array of objects (like a list of contacts or products)
- [ ] You can loop through an array of objects and print formatted output

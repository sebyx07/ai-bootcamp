# Challenge: Variables and Types

## What you'll learn
How to store data in variables and work with different data types: strings, numbers, booleans, null, and undefined.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Focus on the variables and data types sections (around 10-25 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can declare variables using `let` and `const`
- [ ] You can explain why `var` should be avoided
- [ ] You have used strings, numbers, booleans, null, and undefined
- [ ] You have used `typeof` to check the type of a variable
- [ ] You have used template literals (backtick strings) to combine variables and text

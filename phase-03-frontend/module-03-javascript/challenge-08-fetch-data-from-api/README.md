# Challenge: Fetch Data from an API

## What you'll learn
How to use JavaScript to load data from the internet and display it on your web page.

## Watch first
- [Fetch API Introduction](https://www.youtube.com/watch?v=Oive66jrwBs) (20 min) — Traversy Media introduces the Fetch API for making HTTP requests from JavaScript

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have used `fetch()` to get data from a public API
- [ ] The data is displayed on an HTML page (not just in the console)
- [ ] You handle the case when the API request fails
- [ ] You opened the page in a browser and see the live data
- [ ] You can explain what an API is and what JSON looks like

# Challenge: If/Else and Loops

## What you'll learn
How to make decisions with if/else statements and repeat actions with loops.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Focus on the conditionals and loops sections (around 25-40 min mark)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write `if`, `else if`, and `else` statements
- [ ] You have used comparison operators (`===`, `!==`, `>`, `<`, `>=`, `<=`)
- [ ] You can write a `for` loop that counts from 1 to 10
- [ ] You can write a `while` loop
- [ ] You have built a simple program that uses both conditionals and loops

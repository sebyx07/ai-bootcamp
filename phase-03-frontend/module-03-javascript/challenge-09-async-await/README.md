# Challenge: Async/Await

## What you'll learn
How to use `async` and `await` to write cleaner asynchronous code instead of chaining `.then()` calls.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Review the async section near the end of the video

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write an `async` function that uses `await`
- [ ] You have refactored your fetch code from the previous challenge to use async/await
- [ ] You handle errors with `try/catch` instead of `.catch()`
- [ ] You can explain why some code is asynchronous and what that means
- [ ] You have made multiple API calls in sequence (one after another)

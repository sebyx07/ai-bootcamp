# Challenge: Debug This Broken App

## What you'll learn
How to find and fix bugs in a JavaScript application using the browser console, console.log, and debugging skills.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Review any sections you need; this challenge tests everything from the JS module

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Claude has given you a broken JavaScript app
- [ ] You opened it in the browser and found the bugs using the console
- [ ] You fixed all the bugs yourself (Claude only helped with hints)
- [ ] The app works correctly in the browser
- [ ] You can list the bugs you found and explain why they caused problems

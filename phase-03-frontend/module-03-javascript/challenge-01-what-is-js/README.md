# Challenge: What is JavaScript?

## What you'll learn
What JavaScript does, how to run it, and how to write your first lines of code.

## Watch first
- [JavaScript Crash Course For Beginners](https://www.youtube.com/watch?v=hdI2bqOjy3c) (100 min) — Watch the first 15 minutes covering what JS is and how to run it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have run JavaScript in the browser console
- [ ] You have created a `.js` file and linked it to an HTML page
- [ ] You have used `console.log()` to print a message
- [ ] You have used `alert()` to show a popup
- [ ] You can explain what JavaScript does that HTML and CSS cannot

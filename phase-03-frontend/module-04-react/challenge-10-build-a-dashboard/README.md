# Challenge: Build a Dashboard

## What you'll learn
How to bring together everything from this entire phase to build a multi-section data dashboard that fetches and displays real data.

## Watch first
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Review as needed; this is the final capstone challenge

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your dashboard has at least 3 sections showing different data
- [ ] Each section fetches data from a public API
- [ ] You have a navigation sidebar or top bar
- [ ] The dashboard is responsive (works on mobile and desktop)
- [ ] You have loading states for each data section
- [ ] The app uses routing for at least 2 views
- [ ] You are genuinely proud of what you built

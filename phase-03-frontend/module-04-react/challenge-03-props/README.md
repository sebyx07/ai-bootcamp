# Challenge: Props

## What you'll learn
How to pass data into components using props to make them dynamic and reusable.

## Watch first
- [Learn React In 30 Minutes](https://www.youtube.com/watch?v=hQAHSlTtcmY) (30 min) — Focus on the props section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You pass at least 3 different props to a component
- [ ] You render a list of items by mapping over an array and passing props to each
- [ ] You use props to make a component display different content each time it's used
- [ ] You can explain what props are and how they flow (parent to child)

# Challenge: useEffect

## What you'll learn
How to run side effects in React components — like fetching data when a component loads or updating the document title.

## Watch first
- [React Hooks Explained - Fireship](https://www.youtube.com/watch?v=TNhaISOUy6Q) (12 min) — Covers useEffect along with other essential hooks
- [Learn React In 30 Minutes](https://www.youtube.com/watch?v=hQAHSlTtcmY) (30 min) — Review the hooks section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have used `useEffect` to run code when a component first loads
- [ ] You have used `useEffect` with a dependency array to respond to state changes
- [ ] You have updated the document title dynamically using `useEffect`
- [ ] You can explain what a "side effect" is in React
- [ ] You understand the dependency array: `[]`, `[value]`, and no array

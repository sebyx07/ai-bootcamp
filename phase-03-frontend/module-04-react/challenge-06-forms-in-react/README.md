# Challenge: Forms in React

## What you'll learn
How to handle user input in React using controlled components and state.

## Watch first
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Focus on the form handling sections showing controlled components

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a form with at least 3 input fields controlled by state
- [ ] Each input updates state as the user types
- [ ] You handle form submission and display the submitted data
- [ ] You have basic form validation (e.g., required fields, minimum length)
- [ ] You can explain what a "controlled component" is

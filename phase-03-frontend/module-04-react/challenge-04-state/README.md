# Challenge: State

## What you'll learn
How to make React components interactive using the `useState` hook to track and update data.

## Watch first
- [Learn React In 30 Minutes](https://www.youtube.com/watch?v=hQAHSlTtcmY) (30 min) — Focus on the state and useState section
- [React Hooks Explained - Fireship](https://www.youtube.com/watch?v=TNhaISOUy6Q) (12 min) — Fast overview of React hooks including useState

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have used `useState` to track a value
- [ ] You have a button that updates state when clicked
- [ ] The page re-renders automatically when state changes
- [ ] You have built a counter component (increment, decrement, reset)
- [ ] You can explain the difference between props and state

# Challenge: Routing

## What you'll learn
How to add multiple pages to a React app using React Router, without the browser fully reloading.

## Watch first
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Focus on the routing sections showing React Router setup and navigation

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have installed React Router and set up a router
- [ ] You have at least 3 pages (Home, About, Contact) with navigation between them
- [ ] The URL changes when you navigate (without full page reload)
- [ ] You have a navigation bar that highlights the current page
- [ ] You have a 404 "Not Found" page for unknown routes

# Challenge: Your First Component

## What you'll learn
How to create a React component and use it inside your app.

## Watch first
- [Learn React In 30 Minutes](https://www.youtube.com/watch?v=hQAHSlTtcmY) (30 min) — Focus on the component creation section in the first 10 minutes

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created at least 2 custom components in separate files
- [ ] You import and use those components in `App.jsx`
- [ ] Your components return JSX with real content
- [ ] You can explain the difference between a component and a regular HTML element
- [ ] Your component names start with capital letters (and you know why)

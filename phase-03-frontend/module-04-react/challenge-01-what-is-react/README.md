# Challenge: What is React?

## What you'll learn
What React is, why it exists, and how to set up your first React project.

## Watch first
- [Learn React In 30 Minutes](https://www.youtube.com/watch?v=hQAHSlTtcmY) (30 min) — Web Dev Simplified covers the core concepts of React in a fast, clear tutorial
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Traversy Media builds a full React project; watch the first 15 minutes on setup

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what React is and why developers use it instead of plain JavaScript
- [ ] You have a React project created with Vite running on localhost
- [ ] You can see the default React page in your browser
- [ ] You have modified the default page to show your own content
- [ ] You understand what JSX is

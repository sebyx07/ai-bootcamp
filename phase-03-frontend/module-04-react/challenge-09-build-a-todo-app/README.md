# Challenge: Build a Todo App

## What you'll learn
How to combine everything you've learned in React to build a complete, working todo application from scratch.

## Watch first
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Review as needed; this challenge combines all React concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can add new todos with a text input and submit button
- [ ] You can mark todos as complete (with a checkbox or click)
- [ ] You can delete todos
- [ ] Completed todos are visually different (strikethrough, different color, etc.)
- [ ] You can filter todos: All, Active, Completed
- [ ] The todo count updates automatically
- [ ] The app is styled and looks presentable

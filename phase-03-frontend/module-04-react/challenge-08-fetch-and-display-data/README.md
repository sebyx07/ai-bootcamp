# Challenge: Fetch and Display Data

## What you'll learn
How to load data from an API inside a React component and display it with loading and error states.

## Watch first
- [React Crash Course 2024](https://www.youtube.com/watch?v=LDB4uaJ87e0) (75 min) — Focus on the data fetching sections showing how to load and display API data

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You fetch data from a public API when a component mounts
- [ ] You show a loading indicator while data is being fetched
- [ ] You display the data in a styled list or grid of cards
- [ ] You handle and display errors if the fetch fails
- [ ] You have a button to refresh/reload the data

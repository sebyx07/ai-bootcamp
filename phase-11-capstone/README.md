# Phase 09: Capstone

This is it. You have learned the terminal, Git, frontend, backend, testing, infrastructure, advanced Claude Code techniques, and AI fundamentals. Now you build something real.

Choose one of three project options — or propose your own. Each project is designed to take 1-2 weeks and demonstrate everything you have learned. Claude will be your senior colleague, not your teacher. Less hand-holding, more collaboration.

## Project Options

### Option A: Full-Stack App
Build a complete full-stack web application with a React frontend and a Node.js (or Rails) backend. Database, authentication, API, and deployment included.

### Option B: SaaS MVP
Build a minimum viable product for a software-as-a-service business: landing page, user authentication, one core feature, and deployment.

### Option C: Your Own Idea
Propose your own project. Claude will help you scope it, plan it, and build it. The only requirement: it must use skills from at least three phases of the bootcamp.

## How This Phase Works

1. Pick your project option
2. Read the project README for requirements and guidance
3. Open Claude Code — it is configured as a senior colleague, not a tutorial guide
4. Build, ship, and present your work

## Prerequisites

- Completed Phases 01-08

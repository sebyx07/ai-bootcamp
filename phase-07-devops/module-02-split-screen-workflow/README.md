# Module 02: Split Screen Workflow

## What this module is about

This is the most important workflow you will learn in this entire bootcamp. Professional developers work with a split terminal: one side for thinking and planning (Claude Code), the other side for doing (the remote server). This module teaches you that workflow and makes it second nature.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - Terminal Multiplexing** | Learn tmux to split your terminal into panes and windows |
| **02 - Claude on Left, Server on Right** | Set up the professional split-screen workflow |
| **03 - Follow Logs Remotely** | Watch real-time logs from your server while working with Claude |
| **04 - Let Claude Work on Remote** | Discover that Claude can SSH into your server and run commands for you |

## By the end of this module

You will be working with Claude Code on the left and your remote server on the right, switching between them effortlessly. You will also know how to let Claude do complex tasks on the remote server directly.

# Challenge: Claude on Left, Server on Right

## What you'll learn
The professional split-screen workflow: Claude Code on the left pane helping you think, SSH to your server on the right pane where you execute commands.

## Watch first
- [Tmux in 100 Seconds](https://www.youtube.com/watch?v=vtB1J_zCv8I) (2 min) — Quick review of tmux if you need a refresher

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a tmux session with two vertical panes
- [ ] Claude Code is running in the left pane
- [ ] You are SSH'd into your server in the right pane
- [ ] You have asked Claude a question and then executed the answer on the server
- [ ] You can switch between panes without thinking about it
- [ ] You have practiced this workflow for at least 10 minutes

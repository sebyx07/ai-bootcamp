# Challenge: Follow Logs Remotely

## What you'll learn
How to watch real-time logs from your server while working with Claude — a critical debugging skill.

## Watch first
- [Linux Logs Explained](https://www.youtube.com/watch?v=6TEvEimVJzM) (10 min) — Gary Smith explains where Linux logs live and how to read them

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can use `tail -f` to follow a log file in real time
- [ ] You have watched the syslog update while doing things on the server
- [ ] You understand the difference between `cat`, `tail`, and `tail -f`
- [ ] You can follow logs in one tmux pane while running commands in another
- [ ] You know where the main log files live on an Ubuntu server

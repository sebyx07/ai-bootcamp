# Challenge: Let Claude Work on Remote

## What you'll learn
How to let Claude Code SSH into your server and run commands for you — turning Claude into a remote server assistant.

## Watch first
- [Tmux in 100 Seconds](https://www.youtube.com/watch?v=vtB1J_zCv8I) (2 min) — Quick tmux refresher if needed

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have asked Claude to run a command on your remote server and it did
- [ ] Claude has SSH'd into your server and checked something for you
- [ ] You understand when to let Claude work on the server vs when to type commands yourself
- [ ] You have used Claude to do a multi-step task on the server (e.g., install something and configure it)
- [ ] You watched what Claude did and understood every command

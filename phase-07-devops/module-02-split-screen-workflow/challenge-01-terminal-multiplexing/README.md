# Challenge: Terminal Multiplexing

## What you'll learn
How to use tmux to split your terminal into multiple panes and windows, so you can see several things at once.

## Watch first
- [Tmux in 100 Seconds](https://www.youtube.com/watch?v=vtB1J_zCv8I) (2 min) — Fireship's fast overview of what tmux is and why it matters
- [Learn tmux - Part 1](https://www.youtube.com/watch?v=nTqu6w2wc68) (15 min) — Learn Linux TV's beginner guide to getting started with tmux

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have tmux installed on your local machine
- [ ] You can start a tmux session with `tmux`
- [ ] You can split your terminal vertically (left/right) with `Ctrl+b %`
- [ ] You can split your terminal horizontally (top/bottom) with `Ctrl+b "`
- [ ] You can switch between panes with `Ctrl+b arrow-keys`
- [ ] You can detach from a session with `Ctrl+b d` and reattach with `tmux attach`

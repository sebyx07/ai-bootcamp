# Challenge: Check Server Health

## What you'll learn
The first commands to run when you SSH into a server to check if things are working.

## Watch first
- [Linux Server Monitoring Commands](https://www.youtube.com/watch?v=_ao3ssMBD9I) (12 min) — Chris Titus Tech covers essential monitoring commands every Linux admin needs

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can check how long the server has been running with `uptime`
- [ ] You can see who is logged in with `w` or `who`
- [ ] You can check running services with `systemctl list-units --type=service --state=running`
- [ ] You can quickly check if your specific services (nginx, myapp) are running
- [ ] You have a mental "health check" routine you can run in 30 seconds

# Module 03: Server Setup

## What this module is about

You just got a fresh server. It has nothing on it except a basic Linux installation. Your job is to set it up so it can run a web application safely. This module walks you through the entire process: creating users, setting up a firewall, installing a web server, deploying your app, and managing it as a service.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - Users and Permissions** | Create users, understand file permissions, and use sudo properly |
| **02 - Firewall Basics** | Set up UFW to control what traffic can reach your server |
| **03 - Install Nginx on Remote** | Install and configure Nginx on your VM to serve web traffic |
| **04 - Deploy App to Remote** | Get your Node.js app running on the server behind Nginx |
| **05 - systemd Services** | Create a systemd service so your app starts automatically and stays running |

## By the end of this module

Your VM will be running a web application behind Nginx, managed by systemd, protected by a firewall, with proper user permissions. This is a real production-like server setup.

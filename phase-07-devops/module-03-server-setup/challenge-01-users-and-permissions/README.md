# Challenge: Users and Permissions

## What you'll learn
How to create users on a Linux server, understand file permissions, and use sudo properly.

## Watch first
- [Linux File Permissions in 5 Minutes](https://www.youtube.com/watch?v=LnKoncbQBsM) (5 min) — Aaron Jack gives a fast, visual breakdown of Linux permissions
- [Linux Users and Groups](https://www.youtube.com/watch?v=jwnvKOjmtEA) (12 min) — Learn Linux TV explains users, groups, and how they relate to permissions

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a new user on the server with `adduser`
- [ ] You can explain what `sudo` does and when to use it
- [ ] You can read file permissions using `ls -la` (e.g., `drwxr-xr-x`)
- [ ] You can change file ownership with `chown` and permissions with `chmod`
- [ ] You understand the three permission groups: owner, group, others
- [ ] You know why you should not run everything as root

# Challenge: Deploy App to Remote

## What you'll learn
How to get a Node.js application running on your remote server, served through Nginx.

## Watch first
- [How to Deploy a Node.js App to a Linux Server](https://www.youtube.com/watch?v=oykl1Ih9pMg) (25 min) — Traversy Media walks through deploying a Node.js app with Nginx as a reverse proxy
- [NGINX Reverse Proxy Tutorial](https://www.youtube.com/watch?v=lZVAI3PqgHc) (8 min) — Quick explanation of how Nginx forwards requests to your app

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Node.js is installed on your server
- [ ] You have a simple Node.js app running on the server
- [ ] Nginx is configured as a reverse proxy, forwarding requests to your app
- [ ] You can visit your VM's IP in a browser and see your app's response
- [ ] You understand that Nginx handles port 80 and forwards to your app on port 3000

# Challenge: Install Nginx on Remote

## What you'll learn
How to install and configure Nginx on your remote server so it can serve web traffic.

## Watch first
- [NGINX Explained in 100 Seconds](https://www.youtube.com/watch?v=JKxlsvZXG7c) (2 min) — Fireship's quick breakdown of what Nginx does
- [Nginx Tutorial for Beginners](https://www.youtube.com/watch?v=9nyiY-psbMs) (20 min) — Crash course on Nginx installation and configuration

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Nginx is installed and running on your VM
- [ ] You can visit your VM's IP address in a browser and see the Nginx welcome page
- [ ] You know where the Nginx config files are (`/etc/nginx/`)
- [ ] You can start, stop, and restart Nginx with `sudo systemctl start/stop/restart nginx`
- [ ] You can test your Nginx config with `sudo nginx -t`

# Challenge: Firewall Basics

## What you'll learn
How to set up a firewall using UFW to control what network traffic can reach your server.

## Watch first
- [UFW Firewall - Complete Tutorial](https://www.youtube.com/watch?v=-CzvPjZ9hp8) (24 min) — HackerSploit walks through setting up and managing UFW step by step
- [Linux Firewall in 100 Seconds](https://www.youtube.com/watch?v=kgdoVeyoO2E) (2 min) — NetworkChuck gives a rapid-fire overview of why firewalls matter

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have installed and enabled UFW on your server
- [ ] You have allowed SSH traffic (port 22) before enabling the firewall
- [ ] You have allowed HTTP traffic (port 80)
- [ ] You can check firewall status with `sudo ufw status`
- [ ] You understand that a firewall controls which ports are open to the network
- [ ] You know why you must allow SSH before enabling the firewall

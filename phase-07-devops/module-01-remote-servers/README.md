# Module 01: Remote Servers

## What this module is about

A server is just a computer that serves things to other computers. In this module, you will create your own server using a virtual machine, connect to it over SSH, and learn to work on it as if it were a computer in a data center halfway around the world.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - What Is a Server** | Understand what servers are and why they exist |
| **02 - Create a VM with libvirt** | Create a virtual machine that acts as your remote server |
| **03 - SSH into Your Server** | Connect to your server securely from your terminal |
| **04 - Navigate a Remote Server** | Move around the filesystem, find things, read files on a remote machine |
| **05 - Install Things Remotely** | Use apt to install software on your server |
| **06 - Copy Files with SCP** | Transfer files between your machine and the server |

## By the end of this module

You will have a running virtual machine that you can SSH into, navigate, install software on, and transfer files to and from. This is your sandbox for the rest of the phase.

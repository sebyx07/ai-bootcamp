# Challenge: Create a VM with libvirt

## What you'll learn
How to create a virtual machine on your own computer that acts like a remote server.

## Watch first
- [you need to learn Virtual Machines RIGHT NOW!!](https://www.youtube.com/watch?v=wX75Z-4MEoM) (23 min) — NetworkChuck explains what VMs are, why you need them, and how to set one up
- [KVM/QEMU Linux Virtual Machines - Getting Started](https://www.youtube.com/watch?v=BgZHbCDFODk) (15 min) — Learn Linux TV walks through setting up KVM/QEMU virtual machines on Linux

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have libvirt and virt-manager installed on your machine
- [ ] You have created an Ubuntu Server VM using virt-manager or the command line
- [ ] Your VM is running and you can see it in `virsh list`
- [ ] You know the IP address of your VM
- [ ] You understand that this VM is a separate computer running inside your computer

# Challenge: Copy Files with SCP

## What you'll learn
How to transfer files between your local machine and a remote server using SCP (Secure Copy).

## Watch first
- [SCP Command - Transfer Files To and From a Server](https://www.youtube.com/watch?v=fmMg6cyww14) (8 min) — Corey Schafer demonstrates SCP for copying files over SSH
- [Linux SCP Command Explained](https://www.youtube.com/watch?v=YxNOkGk81Mo) (6 min) — Quick walkthrough of SCP syntax and common use cases

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can copy a file from your local machine to the server using `scp`
- [ ] You can copy a file from the server back to your local machine
- [ ] You can copy an entire directory using `scp -r`
- [ ] You understand the syntax: `scp source destination` where either can be `user@host:path`

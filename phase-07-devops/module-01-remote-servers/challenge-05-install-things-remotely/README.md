# Challenge: Install Things Remotely

## What you'll learn
How to install software on a remote server using the apt package manager.

## Watch first
- [Linux Package Management - apt explained](https://www.youtube.com/watch?v=vX3krP6JmOY) (8 min) — Learn Linux TV explains how apt works and how to install, update, and remove packages

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can update the package list with `sudo apt update`
- [ ] You can install a package with `sudo apt install`
- [ ] You have installed at least three useful tools on your server (e.g., htop, tree, curl)
- [ ] You can search for packages with `apt search`
- [ ] You can remove a package with `sudo apt remove`
- [ ] You understand the difference between `apt update` and `apt upgrade`

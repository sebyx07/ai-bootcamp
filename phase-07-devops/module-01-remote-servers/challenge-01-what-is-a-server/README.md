# Challenge: What Is a Server?

## What you'll learn
What a server is, why servers exist, and how they differ from your personal computer.

## Watch first
- [Linux in 100 Seconds](https://www.youtube.com/watch?v=rrB13utjYV4) (3 min) — Fireship's fast overview of Linux, the operating system that runs most servers
- [What is a Server? Servers vs Desktops Explained](https://www.youtube.com/watch?v=UjCDWCeHCzY) (10 min) — PowerCert animated explanation of what servers are and how they work

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a server is in your own words
- [ ] You can name at least three things servers do (serve websites, store files, run databases, etc.)
- [ ] You understand that a server is just a computer running Linux, usually without a screen
- [ ] You know what "remote" means — a server you connect to over a network, not sitting on your desk

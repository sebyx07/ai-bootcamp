# Challenge: SSH into Your Server

## What you'll learn
How to connect to a remote server securely using SSH, the tool every developer uses to access servers.

## Watch first
- [SSH Crash Course - How does SSH work?](https://www.youtube.com/watch?v=hQWRp-FdTpc) (22 min) — Traversy Media explains SSH concepts, key generation, and connecting to servers
- [How SSH Works - Computerphile](https://www.youtube.com/watch?v=ORcvSkgdA58) (14 min) — Clear explanation of how SSH encryption and authentication work under the hood

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can SSH into your VM using `ssh username@ip-address`
- [ ] You have generated an SSH key pair on your local machine
- [ ] You have copied your public key to the server using `ssh-copy-id`
- [ ] You can SSH into your VM without typing a password
- [ ] You understand that the private key stays on your machine and the public key goes on the server

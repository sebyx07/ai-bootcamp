# Module 03: Ruby Testing

## What you will learn

How to use RSpec, the standard Ruby testing framework, to write and run tests for models and controllers in a Rails application.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - First RSpec Test** | Install RSpec, write your first spec, and see it pass |
| **02 - Testing Models** | Write tests for Rails model validations and methods |
| **03 - Testing Controllers** | Write tests for Rails controller actions and responses |

## Why this matters

RSpec is to Ruby what Jest is to JavaScript -- the go-to testing tool. Rails developers write RSpec tests every day. Understanding it will make you a much more employable Ruby developer.

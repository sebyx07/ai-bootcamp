# Challenge: Testing Models

## What you'll learn
How to write RSpec tests for Rails model validations, associations, and custom methods.

## Watch first
- [RSpec Model Testing in Rails](https://www.youtube.com/watch?v=1ySdisNMXpk) (15 min) — Practical guide to testing Rails models with RSpec

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a Rails app with at least one model that has validations
- [ ] RSpec and the `rspec-rails` gem are installed
- [ ] You have tests that verify validations (e.g., presence, uniqueness, length)
- [ ] You have tests for at least one custom model method
- [ ] Running `bundle exec rspec` shows all tests passing

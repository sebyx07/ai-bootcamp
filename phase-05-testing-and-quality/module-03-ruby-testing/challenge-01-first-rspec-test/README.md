# Challenge: First RSpec Test

## What you'll learn
How to install RSpec, write your first Ruby test (called a "spec"), and run it from the command line.

## Watch first
- [RSpec Tutorial for Beginners](https://www.youtube.com/watch?v=Dj19O9kLK6w) (20 min) — Clear walkthrough of RSpec basics with practical examples
- [Introduction to RSpec](https://www.youtube.com/watch?v=yP4Gkr8esi4) (12 min) — Beginner-friendly explanation of RSpec describe, it, and expect syntax

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] RSpec is installed in a Ruby project (via `gem install rspec` or Gemfile)
- [ ] You have run `rspec --init` to set up the spec directory
- [ ] You have a spec file that ends in `_spec.rb`
- [ ] You have written at least 3 tests using `expect().to eq()`
- [ ] Running `rspec` shows all tests passing in green

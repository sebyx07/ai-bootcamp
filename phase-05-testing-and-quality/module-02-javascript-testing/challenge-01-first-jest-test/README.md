# Challenge: First Jest Test

## What you'll learn
How to install Jest, write your very first real test using a testing framework, and run it from the command line.

## Watch first
- [Jest Crash Course - Unit Testing in JavaScript](https://www.youtube.com/watch?v=7r4xVDI2vho) (57 min) — Traversy Media's complete Jest walkthrough (watch the first 15 minutes for setup and basics)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Jest is installed as a dev dependency in a project
- [ ] You have a test file that ends in `.test.js`
- [ ] You have written at least 3 tests using `expect().toBe()`
- [ ] Running `npm test` shows all tests passing with green checkmarks
- [ ] You understand what `describe`, `test`, and `expect` do

# Challenge: Testing API Endpoints

## What you'll learn
How to test Express API routes using Supertest to make sure your endpoints return the right status codes and data.

## Watch first
- [Testing Node.js API with Jest and Supertest](https://www.youtube.com/watch?v=FKnzS_icp20) (25 min) — Hands-on tutorial for testing Express APIs with Jest and Supertest

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a small Express API with at least 3 routes
- [ ] Supertest is installed and configured to work with Jest
- [ ] You have tests that check HTTP status codes (200, 404, etc.)
- [ ] You have tests that check response body content
- [ ] All tests pass when you run `npm test`

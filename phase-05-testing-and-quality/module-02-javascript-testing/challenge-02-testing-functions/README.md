# Challenge: Testing Functions

## What you'll learn
How to write meaningful tests for real-world JavaScript functions, including edge cases and multiple matchers.

## Watch first
- [Jest Crash Course - Unit Testing in JavaScript](https://www.youtube.com/watch?v=7r4xVDI2vho) (57 min) — Continue from where you left off, focusing on matchers and async testing (15:00-35:00)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have tested a string manipulation function with at least 3 test cases
- [ ] You have tested a function that works with arrays or objects
- [ ] You have used matchers beyond `toBe` (like `toEqual`, `toContain`, `toBeTruthy`)
- [ ] You have written at least one test for an edge case (empty input, null, etc.)
- [ ] All tests pass when you run `npm test`

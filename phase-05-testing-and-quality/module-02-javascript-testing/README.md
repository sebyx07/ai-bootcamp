# Module 02: JavaScript Testing

## What you will learn

How to use Jest, the most popular JavaScript testing framework, to write and run automated tests for functions and API endpoints.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - First Jest Test** | Install Jest, write your very first test, and see it pass |
| **02 - Testing Functions** | Write tests for real JavaScript functions with multiple cases |
| **03 - Testing API Endpoints** | Test Express API routes to make sure they return the right data |

## Why this matters

Jest is the standard testing tool in JavaScript. If you work on any JavaScript project, you will encounter Jest tests. Learning it now means you can confidently change code knowing your tests will catch mistakes.

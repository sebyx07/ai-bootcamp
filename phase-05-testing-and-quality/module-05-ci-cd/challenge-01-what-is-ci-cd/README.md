# Challenge: What Is CI/CD?

## What you'll learn
What Continuous Integration and Continuous Deployment mean and why modern teams rely on automated pipelines.

## Watch first
- [DevOps CI/CD Explained in 100 Seconds](https://www.youtube.com/watch?v=scEDHsr3APg) (2 min) — Fireship's fast explanation of CI/CD concepts
- [GitHub Actions Tutorial - Basic Concepts and CI/CD Pipeline with Docker](https://www.youtube.com/watch?v=R8_veQiYBjI) (32 min) — TechWorld with Nana's thorough walkthrough (watch the first 10 minutes for concepts)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain CI (Continuous Integration) in your own words
- [ ] You can explain CD (Continuous Deployment) in your own words
- [ ] You understand what a "pipeline" is in this context
- [ ] You can describe a real-world scenario where CI/CD prevents a bug from reaching users
- [ ] You know what GitHub Actions is and why it is popular

# Challenge: Manual vs Automated Testing

## What you'll learn
The difference between testing by hand and writing automated tests, and why automation wins at scale.

## Watch first
- [Manual vs Automation Testing](https://www.youtube.com/watch?v=GpBqMNg5qjQ) (7 min) — Clear side-by-side comparison of manual and automated testing approaches
- [Test-Driven Development in 100 Seconds](https://www.youtube.com/watch?v=Jv2uxzhPFl4) (2 min) — Fireship's quick look at TDD, a workflow built on automated tests

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have manually tested a small program by running it and checking the output yourself
- [ ] You have written automated tests for the same program
- [ ] You have intentionally broken the code and seen your automated tests catch the bug
- [ ] You can explain when manual testing is appropriate and when automated testing is better

# Challenge: What Is a Test?

## What you'll learn
What a software test actually is and how to think about testing your code.

## Watch first
- [Software Testing Explained in 100 Seconds](https://www.youtube.com/watch?v=u6QfIXgjwGQ) (2 min) — Fireship's rapid-fire explanation of what testing is and why it matters
- [Software Testing Tutorial for Beginners](https://www.youtube.com/watch?v=oLc9gVM8FBM) (8 min) — Guru99's gentle introduction to software testing concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a test is in your own words
- [ ] You have written a test in plain English (e.g., "if I add 2 + 2, I expect 4")
- [ ] You have written that same test as actual code that runs and passes
- [ ] You understand what "expected vs actual" means in testing

# Challenge: RuboCop Setup

## What you'll learn
How to install, configure, and run RuboCop on a Ruby project to enforce consistent code style.

## Watch first
- [RuboCop - Ruby Static Code Analyzer](https://www.youtube.com/watch?v=nGpqwSRyBSk) (12 min) — Practical guide to installing and using RuboCop on Ruby projects

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] RuboCop is installed (`gem install rubocop` or via Gemfile)
- [ ] Running `rubocop` shows a list of offenses in your Ruby code
- [ ] You have a `.rubocop.yml` configuration file
- [ ] You have fixed at least 3 offenses by hand
- [ ] You have used `rubocop -a` to auto-correct remaining issues
- [ ] You understand how to disable a specific cop (rule) in the config

# Module 04: Linting

## What you will learn

What linting is, how to set up ESLint for JavaScript and RuboCop for Ruby, and how to use these tools to find and fix code quality issues automatically.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - What Is Linting?** | Understand what linters do and why teams use them |
| **02 - ESLint Setup** | Install and configure ESLint for a JavaScript project |
| **03 - RuboCop Setup** | Install and configure RuboCop for a Ruby project |
| **04 - Fix All Lint Errors** | Take a messy codebase and clean it up using linting tools |

## Why this matters

Linters enforce consistent code style across a project. When everyone follows the same rules, code is easier to read, review, and maintain. Most professional teams require code to pass linting before it can be merged.

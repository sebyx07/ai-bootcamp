# Challenge: What Is Linting?

## What you'll learn
What code linting is, why teams use linters, and how linters differ from tests.

## Watch first
- [ESLint in 100 Seconds](https://www.youtube.com/watch?v=qhuFviJn-es) (2 min) — Fireship's quick explanation of what ESLint does and why it matters
- [What is a Linter and Why You Need One](https://www.youtube.com/watch?v=HDQXWr5TOnI) (8 min) — Beginner-friendly explanation of linting with real examples

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a linter does in your own words
- [ ] You understand the difference between a linter and a test
- [ ] You have seen examples of linting rules (indentation, semicolons, unused variables)
- [ ] You can explain why consistent code style matters on a team

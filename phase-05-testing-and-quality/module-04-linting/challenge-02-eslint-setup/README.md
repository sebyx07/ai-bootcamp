# Challenge: ESLint Setup

## What you'll learn
How to install, configure, and run ESLint on a JavaScript project.

## Watch first
- [How to Setup ESLint and Prettier](https://www.youtube.com/watch?v=St1YSNoB36Y) (10 min) — Practical walkthrough of setting up ESLint in a real project

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] ESLint is installed in a JavaScript project
- [ ] You have an `eslint.config.js` configuration file
- [ ] Running `npx eslint .` shows linting results
- [ ] You have fixed at least 3 linting errors by hand
- [ ] You have used `npx eslint --fix .` to auto-fix remaining issues
- [ ] You understand what a linting rule is and how to enable/disable one

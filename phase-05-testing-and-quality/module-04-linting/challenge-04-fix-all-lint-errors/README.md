# Challenge: Fix All Lint Errors

## What you'll learn
How to take a messy, poorly-styled codebase and systematically clean it up using linting tools.

## Watch first
- [Clean Code with ESLint](https://www.youtube.com/watch?v=ZXW6Jn6or1w) (12 min) — Practical demonstration of using ESLint to clean up a messy project

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have received a messy JavaScript file AND a messy Ruby file from Claude
- [ ] You have run ESLint on the JavaScript file and fixed ALL errors
- [ ] You have run RuboCop on the Ruby file and fixed ALL offenses
- [ ] Both files pass linting with zero warnings and zero errors
- [ ] The code still works correctly after cleanup (tests pass if any exist)
- [ ] You did NOT just auto-fix everything -- you fixed at least half by hand

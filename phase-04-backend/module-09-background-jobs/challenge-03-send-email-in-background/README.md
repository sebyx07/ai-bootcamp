# Challenge: Send Email in Background

## What you'll learn
How to send emails asynchronously using Action Mailer and Sidekiq.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Reference for Action Mailer concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a mailer in Rails
- [ ] You can send an email using a background job
- [ ] You can see the email in the Rails development log (or Letter Opener)
- [ ] You understand why emails should always be sent in the background

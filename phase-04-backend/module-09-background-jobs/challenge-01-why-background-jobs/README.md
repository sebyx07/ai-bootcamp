# Challenge: Why Background Jobs

## What you'll learn
Why some tasks should not run during a web request and how background jobs solve this problem.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (3 min) — Redis powers background job queues like Sidekiq

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain why sending an email during a web request is a bad idea
- [ ] You can name 3 tasks that should be background jobs
- [ ] You understand the concept of a job queue
- [ ] You know what Sidekiq is and that it uses Redis

# Challenge: Sidekiq Basics

## What you'll learn
How to set up Sidekiq and create your first background job in Rails.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (3 min) — Reminder that Sidekiq depends on Redis

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have Sidekiq installed and running
- [ ] You have created a background job class
- [ ] You can enqueue a job and see it processed by Sidekiq
- [ ] You can see the Sidekiq dashboard (web UI)

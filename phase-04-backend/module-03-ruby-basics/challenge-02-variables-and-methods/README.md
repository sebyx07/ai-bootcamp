# Challenge: Variables and Methods

## What you'll learn
How Ruby handles variables, data types, strings, and method definitions.

## Watch first
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (3 min) — Rewatch if needed for Ruby syntax overview

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can declare variables and use different data types (strings, numbers, arrays, hashes)
- [ ] You can define and call methods with arguments
- [ ] You understand Ruby string interpolation (`"Hello, #{name}"`)
- [ ] You know what a Ruby hash is and how it compares to a JavaScript object

# Challenge: Reading Ruby Code

## What you'll learn
How to read and understand existing Ruby code -- a critical skill for working with Rails and open-source projects.

## Watch first
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (3 min) — Final review of Ruby concepts before reading real code

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have read 3-4 Ruby code snippets and explained what they do
- [ ] You can trace through Ruby code step by step
- [ ] You can identify classes, methods, blocks, and iterators in unfamiliar code
- [ ] You feel confident reading Ruby code you did not write

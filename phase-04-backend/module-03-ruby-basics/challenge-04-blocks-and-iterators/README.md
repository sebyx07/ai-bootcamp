# Challenge: Blocks and Iterators

## What you'll learn
How to use blocks, `each`, `map`, and `select` -- Ruby's powerful way of working with collections.

## Watch first
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (3 min) — Revisit for blocks syntax

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can use `.each` to loop through an array
- [ ] You can use `.map` to transform an array
- [ ] You can use `.select` to filter an array
- [ ] You understand the difference between `do...end` blocks and `{ }` blocks

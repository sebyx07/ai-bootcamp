# Challenge: IRB Hello World

## What you'll learn
How to run Ruby code interactively using IRB (Interactive Ruby) and write your first Ruby program.

## Watch first
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (3 min) — Fireship gives a fast overview of what Ruby is and why developers love it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have opened IRB and run Ruby code interactively
- [ ] You have printed "Hello World" in Ruby
- [ ] You have done basic math and string operations in IRB
- [ ] You have created and run a `.rb` file from the terminal

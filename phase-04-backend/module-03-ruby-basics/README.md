# Module 03: Ruby Basics

Ruby is a programming language designed to make programmers happy. You are learning just enough Ruby to be productive with Ruby on Rails. If you know JavaScript, Ruby will feel familiar but cleaner.

## What you will learn

- How to run Ruby code in IRB (Interactive Ruby)
- Variables, methods, and basic Ruby syntax
- Classes and objects
- Blocks and iterators (Ruby's superpower)
- How to read and understand existing Ruby code

## Challenges

| # | Challenge | What you will do |
|---|-----------|-----------------|
| 1 | IRB Hello World | Run your first Ruby code in the interactive console |
| 2 | Variables and Methods | Learn Ruby syntax for variables, strings, and methods |
| 3 | Classes | Create Ruby classes with attributes and methods |
| 4 | Blocks and Iterators | Use blocks, each, map, and select |
| 5 | Reading Ruby Code | Read existing Ruby code and explain what it does |

## Why this matters

Ruby on Rails is written in Ruby. You do not need to be a Ruby expert to use Rails, but you need to understand the basics: how methods work, what classes are, and how to read Ruby code when you see it.

# Challenge: Classes

## What you'll learn
How to create Ruby classes with attributes, methods, and initialize constructors.

## Watch first
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (3 min) — Quick reminder of Ruby's object-oriented nature

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can create a Ruby class with an `initialize` method
- [ ] You understand `attr_reader`, `attr_writer`, and `attr_accessor`
- [ ] You can create instances of your class and call methods on them
- [ ] You have built a class that represents something real (a Dog, a Book, a Player, etc.)

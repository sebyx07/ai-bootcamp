# Challenge: Key-Value Operations

## What you'll learn
How to use Redis commands to store, retrieve, and manage data.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (3 min) — Rewatch for the commands overview

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can use SET, GET, DEL, and EXISTS commands
- [ ] You can set expiration times on keys (TTL)
- [ ] You can use lists and hashes in Redis
- [ ] You understand how Redis data types differ from SQL tables

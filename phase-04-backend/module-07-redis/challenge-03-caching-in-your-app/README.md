# Challenge: Caching in Your App

## What you'll learn
How to use Redis as a cache in your Rails application to speed up slow operations.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (3 min) — Quick reminder of Redis caching use case

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand what caching is and when to use it
- [ ] You have connected Redis to your Rails app
- [ ] You have cached at least one expensive operation
- [ ] You can see the performance difference with and without caching

# Challenge: What Is Redis

## What you'll learn
What Redis is, how it differs from a traditional database, and why it is so fast.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (3 min) — Fireship explains Redis and why it is blazing fast

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what Redis is in your own words
- [ ] You know why Redis is faster than a regular database
- [ ] You can name 3 use cases for Redis (caching, sessions, queues)
- [ ] You have Redis running locally and have connected to it

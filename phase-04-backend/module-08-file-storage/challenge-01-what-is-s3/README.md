# Challenge: What Is S3

## What you'll learn
What object storage (S3) is, why you need it, and how to run it locally with LocalStack (open-source).

## Watch first
- [Fireship — S3 in 100 Seconds](https://www.youtube.com/watch?v=77lMCiiMilo) (2 min) — quick overview of what S3 is
- [TechWorld with Nana — AWS S3 Tutorial](https://www.youtube.com/watch?v=e6w9LwZJFIA) (12 min) — understand buckets, objects, and how S3 works

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what S3 is and what "object storage" means
- [ ] You understand why you should not store uploaded files in your database
- [ ] You know what buckets and objects are in S3
- [ ] You have LocalStack running locally via Docker as your S3 replacement
- [ ] You understand that LocalStack is an open-source local replacement for AWS services

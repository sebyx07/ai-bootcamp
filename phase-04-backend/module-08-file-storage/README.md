# Module 08: File Storage

When users upload images, documents, or videos, where do those files go? You cannot store large files in your database. Services like Amazon S3 provide scalable, reliable file storage in the cloud.

## What you will learn

- What S3 is and how cloud file storage works
- How to upload files from your application
- How to retrieve and display stored files

## Challenges

| # | Challenge | What you will do |
|---|-----------|-----------------|
| 1 | What Is S3 | Understand cloud file storage and how S3 works |
| 2 | Upload and Retrieve | Upload a file and retrieve it in your application |

## Why this matters

Almost every application needs to handle file uploads at some point -- profile photos, attachments, documents. Understanding cloud storage is essential for building real applications.

# Challenge: Upload and Retrieve

## What you'll learn
How to add file upload functionality to your Rails application using Active Storage.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Reference for Rails features

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have set up Active Storage in your Rails app
- [ ] Active Storage is configured to use LocalStack (your local S3)
- [ ] Users can upload an image when creating an article
- [ ] Uploaded images are stored in LocalStack and displayed on the article page
- [ ] You understand how to switch from LocalStack to real S3 in production

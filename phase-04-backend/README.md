# Phase 04: Backend

You have been building things people can see -- HTML pages, styled components, interactive JavaScript. Now you will learn what happens behind the scenes. When you click "Log in" or "Post a comment," where does that data go? How does a server know what to send back?

This phase takes you from zero backend knowledge to building a full blog application with a database, user login, and more.

## What you will learn

- How the web actually works (requests, responses, HTTP)
- Building servers with Node.js and Express
- The Ruby programming language (enough to use Rails)
- Ruby on Rails -- a powerful framework for building web apps fast
- Databases and SQL
- User authentication (login/signup)
- Redis, file storage (S3), and background jobs

## Prerequisites

- Completed Phase 03 (Frontend) or comfortable with HTML, CSS, and JavaScript
- Terminal basics from Phase 01
- Git basics from Phase 01

## How to work through this phase

1. Start with Module 01 -- do not skip it, even if you think you know how the web works
2. Module 02 (Node/Express) gives you your first backend with JavaScript you already know
3. Modules 03-04 switch to Ruby and Rails -- this is the big section
4. Modules 05-06 add databases and login -- essential for any real app
5. Modules 07-09 are shorter exposure modules -- you will learn what these tools are and try basic usage

## By the end of this phase

You will have built a blog application with Ruby on Rails that has:
- Posts that are stored in a database
- User signup and login
- The ability to create, edit, and delete posts
- An understanding of how all the pieces fit together

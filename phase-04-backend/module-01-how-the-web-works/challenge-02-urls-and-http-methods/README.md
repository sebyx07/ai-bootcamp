# Challenge: URLs and HTTP Methods

## What you'll learn
How URLs are structured and what the four main HTTP methods (GET, POST, PUT, DELETE) mean.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — If you have not watched it yet; focus on the section about HTTP methods

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can break a URL into its parts (protocol, domain, path, query string)
- [ ] You can explain what GET, POST, PUT, and DELETE are used for
- [ ] You have sent requests with different methods using curl
- [ ] You understand when to use each method

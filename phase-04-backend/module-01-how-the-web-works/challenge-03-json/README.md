# Challenge: JSON

## What you'll learn
How to read and write JSON -- the data format that servers and clients use to communicate.

## Watch first
- [Learn JSON in 10 Minutes](https://www.youtube.com/watch?v=iiADhChRriM) (10 min) — Web Dev Simplified explains JSON syntax, why it matters, and how to use it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what JSON is and why it is used
- [ ] You can write valid JSON by hand
- [ ] You can spot errors in broken JSON
- [ ] You know the difference between JSON and a JavaScript object

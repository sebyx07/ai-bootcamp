# Challenge: Request and Response

## What you'll learn
How the HTTP request/response cycle works -- what your browser sends and what the server sends back.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — Traversy Media covers HTTP fundamentals, the request/response cycle, status codes, and headers

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what happens when you type a URL and press Enter
- [ ] You have used browser dev tools (Network tab) to see real HTTP requests
- [ ] You can identify the request method, URL, status code, and response body
- [ ] You understand the difference between a request and a response

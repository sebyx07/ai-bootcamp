# Challenge: Use curl and Postman

## What you'll learn
How to use curl and Postman to send HTTP requests to real APIs -- essential tools for backend development.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — Rewatch the Postman sections if needed; Traversy uses Postman throughout

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can send GET, POST, PUT, and DELETE requests with curl
- [ ] You can send requests with JSON body data using curl
- [ ] You can set custom headers in curl
- [ ] You have explored a public API (like JSONPlaceholder) using curl

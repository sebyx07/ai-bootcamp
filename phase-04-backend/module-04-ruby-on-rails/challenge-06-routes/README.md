# Challenge: Routes

## What you'll learn
How Rails routing maps URLs to controller actions and how to define custom routes.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the routing sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand what `resources :articles` generates (all 7 routes)
- [ ] You can run `rails routes` and read the output
- [ ] You can add custom routes
- [ ] You understand the difference between `resources`, `resource`, and custom `get`/`post` routes

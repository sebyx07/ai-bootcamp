# Challenge: Controllers

## What you'll learn
How Rails controllers handle incoming requests, interact with models, and prepare data for views.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the controllers section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a controller does in MVC
- [ ] You can write controller actions (index, show, new, create, edit, update, destroy)
- [ ] You understand how the controller connects routes to models and views
- [ ] You have written a controller action from scratch (not generated)

# Challenge: Forms and Strong Params

## What you'll learn
How to build HTML forms in Rails and securely handle the data they submit using strong parameters.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the forms section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can build a form using Rails form helpers (`form_with`)
- [ ] You understand what strong parameters are and why they exist
- [ ] You can handle form submission in a controller
- [ ] You can display validation errors in the form

# Challenge: Associations

## What you'll learn
How to connect models together -- like articles having many comments, or users having many posts.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the associations/relationships sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain `has_many` and `belongs_to`
- [ ] You have created an association between two models
- [ ] You can access associated records (e.g., `article.comments`)
- [ ] You understand what a foreign key is

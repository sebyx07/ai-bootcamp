# Challenge: Models

## What you'll learn
How Rails models represent database tables and how to use them to create, read, update, and delete records.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the models/database sections (around 45 min in)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can use the Rails console to create, read, update, and delete records
- [ ] You understand what ActiveRecord is
- [ ] You can use methods like `.all`, `.find`, `.create`, `.update`, and `.destroy`
- [ ] You understand the connection between a model and a database table

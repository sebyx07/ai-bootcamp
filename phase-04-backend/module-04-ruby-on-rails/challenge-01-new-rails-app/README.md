# Challenge: New Rails App

## What you'll learn
How to create a brand new Ruby on Rails application and understand its folder structure.

## Watch first
- [Ruby on Rails in 100 Seconds](https://www.youtube.com/watch?v=2DvrRadXwWY) (3 min) — Fireship explains what Rails is and why it is powerful
- [DHH Rails 7 Demo](https://www.youtube.com/watch?v=mpWFrUwAN88) (35 min) — The creator of Rails shows how fast you can build with it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a new Rails app with `rails new`
- [ ] You can start the Rails server and see the welcome page
- [ ] You can name at least 5 important folders in a Rails app and what they are for
- [ ] You understand what "convention over configuration" means

# Challenge: Views and ERB

## What you'll learn
How to build HTML views in Rails using ERB (Embedded Ruby) templates.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the views/ERB sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write ERB templates with `<%= %>` and `<% %>`
- [ ] You understand the difference between `<%= %>` (output) and `<% %>` (execute)
- [ ] You can loop through data in a view
- [ ] You understand layouts and partials

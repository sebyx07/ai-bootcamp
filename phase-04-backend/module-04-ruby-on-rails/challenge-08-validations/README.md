# Challenge: Validations

## What you'll learn
How to add rules to your models so that bad data cannot be saved to the database.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the validations section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can add presence, length, and uniqueness validations
- [ ] You can check if a record is valid and see error messages
- [ ] You understand when validations run (before save)
- [ ] You have added validations to your Article and Comment models

# Challenge: Build a Blog

## What you'll learn
How to put everything together by building a complete blog application from scratch.

## Watch first
- [Ruby on Rails in 100 Seconds](https://www.youtube.com/watch?v=2DvrRadXwWY) (3 min) — Quick motivation before the capstone
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Reference for any concepts you need to revisit

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a blog with Articles that have a title, body, and published date
- [ ] Articles have many Comments with author and body
- [ ] Both models have proper validations
- [ ] You can create, read, update, and delete articles through the browser
- [ ] You can add comments to articles
- [ ] The blog has a homepage that lists all articles
- [ ] You built most of it yourself (not just running generators)

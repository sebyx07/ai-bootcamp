# Challenge: Scaffold and Explore

## What you'll learn
How Rails generators can create an entire CRUD feature in seconds, and what all the generated files do.

## Watch first
- [Ruby on Rails in 100 Seconds](https://www.youtube.com/watch?v=2DvrRadXwWY) (3 min) — Rewatch to see scaffold in action

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have run a Rails scaffold generator
- [ ] You can use the generated CRUD interface in the browser
- [ ] You can identify the model, controller, views, and migration that were generated
- [ ] You understand that scaffolds are learning tools, not production code

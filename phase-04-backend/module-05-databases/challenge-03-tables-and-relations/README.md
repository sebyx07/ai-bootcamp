# Challenge: Tables and Relations

## What you'll learn
How tables relate to each other using foreign keys, and how to query across related tables with JOIN.

## Watch first
- [SQL Tutorial - Full Database Course for Beginners](https://www.youtube.com/watch?v=HXV3zeQKqGY) (4 hrs) — Watch the sections on JOINs and foreign keys

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand what a foreign key is
- [ ] You can write a JOIN query to get data from two tables
- [ ] You understand the relationship between the articles and comments tables
- [ ] You can create a new table with a foreign key using SQL

# Challenge: SQL SELECT, INSERT, UPDATE

## What you'll learn
How to write basic SQL queries to read, create, update, and delete data directly in the database.

## Watch first
- [SQL Tutorial - Full Database Course for Beginners](https://www.youtube.com/watch?v=HXV3zeQKqGY) (4 hrs) — Watch the SELECT, INSERT, UPDATE, DELETE sections (about 30 min total)

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write a SELECT query to read data
- [ ] You can write an INSERT query to add data
- [ ] You can write an UPDATE query to modify data
- [ ] You can write a DELETE query to remove data
- [ ] You can use WHERE to filter results

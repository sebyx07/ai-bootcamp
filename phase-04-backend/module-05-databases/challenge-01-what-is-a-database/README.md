# Challenge: What Is a Database

## What you'll learn
What databases are, why they exist, and how data is organized in tables with rows and columns.

## Watch first
- [SQL Explained in 100 Seconds](https://www.youtube.com/watch?v=zsjvFFKOm3c) (2 min) — Fireship gives a fast overview of SQL and relational databases
- [SQL Tutorial - Full Database Course for Beginners](https://www.youtube.com/watch?v=HXV3zeQKqGY) (4 hrs) — freeCodeCamp full SQL course; watch the first 30 minutes on database concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a database is in your own words
- [ ] You understand tables, rows, and columns
- [ ] You know the difference between SQL and NoSQL databases
- [ ] You have opened a SQLite database and looked at its structure

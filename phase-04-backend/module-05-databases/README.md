# Module 05: Databases

Every real application needs to store data somewhere. Databases are organized systems for storing, retrieving, and managing data. In this module you will learn SQL (the language databases speak) and how to use databases from your applications.

## What you will learn

- What databases are and why we use them
- SQL basics: SELECT, INSERT, UPDATE, DELETE
- Tables, columns, and relationships between tables
- How migrations manage database changes over time
- How to seed a database with initial data
- How to query the database from your application code

## Challenges

| # | Challenge | What you will do |
|---|-----------|-----------------|
| 1 | What Is a Database | Understand what databases are and explore one |
| 2 | SQL SELECT, INSERT, UPDATE | Write basic SQL queries to read and write data |
| 3 | Tables and Relations | Create related tables and understand foreign keys |
| 4 | Migrations | Use migrations to change your database structure safely |
| 5 | Seeds | Populate your database with sample data |
| 6 | Query from Your App | Connect your application to a database and run queries |

## Why this matters

Without a database, your data disappears when the server restarts. Databases are the foundation of every real application -- user accounts, posts, products, orders -- all stored in databases.

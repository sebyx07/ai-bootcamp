# Challenge: Migrations

## What you'll learn
How Rails migrations let you change your database structure safely and track those changes over time.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the migrations section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can create a migration to add a column to an existing table
- [ ] You can run and rollback migrations
- [ ] You understand the `db/schema.rb` file
- [ ] You know why migrations exist instead of editing the database directly

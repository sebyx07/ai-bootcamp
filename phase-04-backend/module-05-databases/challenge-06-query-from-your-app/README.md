# Challenge: Query from Your App

## What you'll learn
How to write more advanced database queries in your Rails application using ActiveRecord and scopes.

## Watch first
- [SQL Explained in 100 Seconds](https://www.youtube.com/watch?v=zsjvFFKOm3c) (2 min) — Quick review of SQL concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can write ActiveRecord queries with conditions, ordering, and limits
- [ ] You have created a scope on a model
- [ ] You can query across associations (e.g., articles with recent comments)
- [ ] You understand the N+1 query problem and eager loading

# Challenge: Seeds

## What you'll learn
How to populate your database with sample data using Rails seed files.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch the seeds section

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have written a seed file that creates sample articles and comments
- [ ] You can run `rails db:seed` to populate your database
- [ ] You can reset and re-seed your database
- [ ] You understand when seeds are useful (development, demos, testing)

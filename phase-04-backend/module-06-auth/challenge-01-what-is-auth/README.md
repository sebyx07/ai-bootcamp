# Challenge: What Is Auth

## What you'll learn
The difference between authentication (who are you?) and authorization (what can you do?), and why both matter.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — Review the sections on HTTP being stateless -- this is why auth is needed

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain the difference between authentication and authorization
- [ ] You understand why HTTP being "stateless" makes auth challenging
- [ ] You can name 3 common authentication methods (passwords, OAuth, API keys)
- [ ] You understand why security matters for user data

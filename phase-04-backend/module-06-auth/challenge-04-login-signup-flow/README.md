# Challenge: Login/Signup Flow

## What you'll learn
How to build actual signup and login functionality for your Rails application.

## Watch first
- [freeCodeCamp Ruby on Rails Full Course](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 hrs) — Watch authentication sections if available

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Users can sign up with a name, email, and password
- [ ] Users can log in with their email and password
- [ ] Users can log out
- [ ] The app shows different content for logged-in vs logged-out users
- [ ] Only logged-in users can create or edit articles

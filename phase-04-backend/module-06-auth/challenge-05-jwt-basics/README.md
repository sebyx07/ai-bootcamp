# Challenge: JWT Basics

## What you'll learn
What JSON Web Tokens (JWTs) are, how they work, and when they are used instead of sessions.

## Watch first
- [Learn JSON in 10 Minutes](https://www.youtube.com/watch?v=iiADhChRriM) (10 min) — Review JSON format since JWTs are JSON-based

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a JWT is and its three parts (header, payload, signature)
- [ ] You understand the difference between session-based auth and token-based auth
- [ ] You have decoded a JWT and seen what is inside
- [ ] You know when to use JWTs vs sessions

# Challenge: Password Hashing

## What you'll learn
Why passwords must be hashed (never stored as plain text) and how hashing works.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — Context for why security matters in HTTP

## You're done when
- [ ] You can explain why storing plain-text passwords is dangerous
- [ ] You understand what hashing is and how it differs from encryption
- [ ] You know what bcrypt is and why it is used
- [ ] You have seen `has_secure_password` in Rails

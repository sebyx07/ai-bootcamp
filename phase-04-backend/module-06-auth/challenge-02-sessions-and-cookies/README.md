# Challenge: Sessions and Cookies

## What you'll learn
How cookies and sessions work together to keep users logged in between requests.

## Watch first
- [HTTP Crash Course & Exploration](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (38 min) — Review the cookies/headers sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a cookie is and how it is sent between browser and server
- [ ] You understand what a session is and where session data is stored
- [ ] You have seen cookies in your browser dev tools
- [ ] You understand the flow: login -> server creates session -> browser stores cookie -> browser sends cookie on every request

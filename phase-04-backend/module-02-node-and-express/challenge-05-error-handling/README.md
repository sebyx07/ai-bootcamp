# Challenge: Error Handling

## What you'll learn
How to handle errors properly in Express so your API does not crash and gives helpful error messages.

## Watch first
- [Learn Express JS in 35 Minutes](https://www.youtube.com/watch?v=SccSCuHhOw0) (35 min) — Web Dev Simplified covers error handling patterns in Express

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a custom error-handling middleware
- [ ] Your API returns proper error messages (not stack traces) for bad requests
- [ ] You handle 404 (not found) routes with a custom handler
- [ ] You understand the difference between operational errors and programmer errors

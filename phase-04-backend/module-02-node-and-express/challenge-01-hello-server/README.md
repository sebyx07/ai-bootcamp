# Challenge: Hello Server

## What you'll learn
How to create your very first web server using Node.js and Express that responds to HTTP requests.

## Watch first
- [Node.js Crash Course 2024](https://www.youtube.com/watch?v=32M1al-Y6Ag) (2 hrs) — Traversy Media covers Node.js fundamentals; watch the first 30 minutes on setup and basics
- [Learn Express JS in 35 Minutes](https://www.youtube.com/watch?v=SccSCuHhOw0) (35 min) — Web Dev Simplified builds an Express server from scratch

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created an Express server that listens on a port
- [ ] Visiting localhost in your browser shows a response from your server
- [ ] You understand what `app.listen()` does
- [ ] You can stop and restart your server

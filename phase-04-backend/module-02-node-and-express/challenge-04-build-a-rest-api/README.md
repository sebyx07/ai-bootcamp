# Challenge: Build a REST API

## What you'll learn
How to build a complete REST API with CRUD operations (Create, Read, Update, Delete).

## Watch first
- [Node.js Crash Course 2024](https://www.youtube.com/watch?v=32M1al-Y6Ag) (2 hrs) — Traversy Media builds a full Node.js API; watch the API-building sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a REST API with all CRUD operations for a resource (e.g., todos, books, movies)
- [ ] GET /items returns all items
- [ ] GET /items/:id returns one item
- [ ] POST /items creates a new item
- [ ] PUT /items/:id updates an item
- [ ] DELETE /items/:id deletes an item
- [ ] You have tested every endpoint with curl

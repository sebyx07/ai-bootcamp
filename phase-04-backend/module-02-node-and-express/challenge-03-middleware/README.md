# Challenge: Middleware

## What you'll learn
What middleware is in Express -- functions that run between receiving a request and sending a response.

## Watch first
- [Learn Express JS in 35 Minutes](https://www.youtube.com/watch?v=SccSCuHhOw0) (35 min) — Web Dev Simplified explains middleware clearly with examples

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what middleware is in your own words
- [ ] You have written a custom logging middleware
- [ ] You understand what `next()` does and why it matters
- [ ] You know the difference between application-level and route-level middleware

# Challenge: Routes

## What you'll learn
How to create multiple routes in Express so different URLs return different responses.

## Watch first
- [Learn Express JS in 35 Minutes](https://www.youtube.com/watch?v=SccSCuHhOw0) (35 min) — Web Dev Simplified covers routing in depth

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your server has at least 4 different routes (GET, POST, etc.)
- [ ] You have routes with URL parameters (like `/users/:id`)
- [ ] You can return JSON responses
- [ ] You understand the difference between route parameters and query strings

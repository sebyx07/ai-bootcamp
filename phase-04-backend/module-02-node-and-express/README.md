# Module 02: Node.js and Express

You already know JavaScript from the frontend. Now you will use it to build a server. Node.js lets you run JavaScript outside the browser, and Express is a framework that makes building web servers simple.

## What you will learn

- How to create a basic web server with Node.js and Express
- How routes work (mapping URLs to code)
- What middleware is and why it matters
- How to build a REST API
- How to handle errors properly

## Challenges

| # | Challenge | What you will do |
|---|-----------|-----------------|
| 1 | Hello Server | Create your first Express server that responds to requests |
| 2 | Routes | Add multiple routes that handle different URLs |
| 3 | Middleware | Understand and use middleware functions |
| 4 | Build a REST API | Build a working CRUD API for a resource |
| 5 | Error Handling | Add proper error handling to your API |

## Why this matters

Node.js and Express are one of the most popular backend combinations in the world. Even though you will learn Rails next, understanding Express gives you a second backend tool and proves that backend development is just JavaScript you already know.

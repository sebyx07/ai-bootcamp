# Challenge: What Is DNS?

## What you'll learn
How the Domain Name System works -- how typing a domain name in your browser leads to a specific server on the internet.

## Watch first
- [DNS Explained in 100 Seconds](https://www.youtube.com/watch?v=UVR9lhUGAyU) (2 min) — Fireship's quick breakdown of how DNS works
- [How DNS Works - Computerphile](https://www.youtube.com/watch?v=uOfonONtIuk) (6 min) — Clear visual explanation of the DNS lookup process

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what DNS does in your own words (the "phone book of the internet")
- [ ] You understand what happens step by step when you type a URL in your browser
- [ ] You know what an A record, CNAME record, and NS record are
- [ ] You have used `nslookup` or `dig` to look up the IP address of a real domain
- [ ] You understand what a domain registrar is (e.g., Namecheap, Google Domains)

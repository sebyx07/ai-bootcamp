# Module 03: Domains and SSL

## What you will learn

How the Domain Name System (DNS) translates website names into server addresses, and how HTTPS uses SSL/TLS certificates to encrypt web traffic.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - What Is DNS?** | Understand how typing a domain name leads to a web server |
| **02 - HTTPS with Let's Encrypt** | Understand what HTTPS is and how free certificates from Let's Encrypt secure websites |

## Why this matters

Every time someone types a URL into their browser, DNS and SSL are working behind the scenes. Understanding these systems helps you debug connection problems, set up websites properly, and keep your users safe.

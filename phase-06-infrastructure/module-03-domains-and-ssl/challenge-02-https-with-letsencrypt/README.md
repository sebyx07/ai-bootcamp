# Challenge: HTTPS with Let's Encrypt

## What you'll learn
What HTTPS is, why every website needs it, and how Let's Encrypt provides free SSL/TLS certificates to secure web traffic.

## Watch first
- [SSL, TLS, HTTPS Explained](https://www.youtube.com/watch?v=j9QmMEWmcfo) (5 min) — Clear beginner-friendly explanation of how HTTPS encryption works
- [How to Setup Free SSL Certificate with Let's Encrypt](https://www.youtube.com/watch?v=ghZXFyIyK1o) (10 min) — Practical walkthrough of getting a free SSL certificate with Certbot

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what HTTPS does and why it matters
- [ ] You understand the difference between HTTP and HTTPS
- [ ] You know what an SSL/TLS certificate is and what it proves
- [ ] You can explain what Let's Encrypt is and why it is important
- [ ] You understand the basic steps to set up HTTPS with Certbot and Nginx
- [ ] You know that certificates expire and must be renewed

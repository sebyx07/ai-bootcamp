# Challenge: Serve Static Files

## What you'll learn
How to configure Nginx to serve a static HTML/CSS website from a directory on your machine.

## Watch first
- [Nginx Tutorial for Beginners](https://www.youtube.com/watch?v=9nyiY-psbMs) (20 min) — Focus on the static file serving sections of this crash course

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a simple HTML/CSS website in a directory (e.g., `/var/www/mysite`)
- [ ] You have written an Nginx server block that serves this website
- [ ] Visiting `http://localhost` shows your website (not the default Nginx page)
- [ ] You understand what `root`, `index`, and `location` directives do
- [ ] You have tested your config with `nginx -t` before restarting

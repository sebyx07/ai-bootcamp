# Module 01: Nginx

## What you will learn

What Nginx is, why it is one of the most widely used web servers in the world, and how to configure it to serve static files and act as a reverse proxy for your applications.

## Challenges

| Challenge | What you will do |
|-----------|-----------------|
| **01 - What Is Nginx?** | Understand what a web server does and where Nginx fits in the picture |
| **02 - Serve Static Files** | Configure Nginx to serve an HTML/CSS website from a directory |
| **03 - Reverse Proxy** | Configure Nginx to forward requests to a Node.js or Rails app running on a different port |

## Why this matters

Almost every website you visit passes through a web server like Nginx. It handles traffic, serves files, and routes requests to backend applications. Understanding Nginx means understanding how the internet actually delivers web pages to browsers.

# Challenge: What Is Nginx?

## What you'll learn
What a web server is, what Nginx does, and why almost every production website uses it.

## Watch first
- [NGINX Explained in 100 Seconds](https://www.youtube.com/watch?v=JKxlsvZXG7c) (2 min) — Fireship's quick breakdown of Nginx as web server, reverse proxy, and load balancer
- [Nginx Tutorial for Beginners](https://www.youtube.com/watch?v=9nyiY-psbMs) (20 min) — Crash course covering Nginx basics, installation, and core concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a web server does in your own words
- [ ] You can explain the difference between Nginx and your Node.js/Rails app
- [ ] You have installed Nginx locally and visited the default welcome page
- [ ] You know what the main Nginx config file is and where it lives
- [ ] You can start, stop, and restart Nginx from the command line

# Challenge: Deploy a Node App

## What you'll learn
The complete process for deploying a Node.js application using PM2 as a process manager and Nginx as a reverse proxy.

## Watch first
- [Full Node.js Deployment - NGINX, SSL With Lets Encrypt](https://www.youtube.com/watch?v=oykl1Ih9SQI) (30 min) — Traversy Media's step-by-step Node.js deployment tutorial
- [Node.js App Deploy on Linux](https://www.youtube.com/watch?v=CvofiEEm6Hg) (15 min) — Quick walkthrough of deploying a Node app to a Linux server

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand the full deployment flow: server setup, code transfer, dependencies, process manager, reverse proxy
- [ ] You know what PM2 is and why it is used instead of `node app.js`
- [ ] You have either deployed to a real server OR walked through the process locally step by step
- [ ] You can explain what happens when a user visits your deployed app
- [ ] You understand how Nginx and PM2 work together

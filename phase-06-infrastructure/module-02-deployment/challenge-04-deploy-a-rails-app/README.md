# Challenge: Deploy a Rails App

## What you'll learn
The complete process for deploying a Ruby on Rails application using Puma as the app server and Nginx as a reverse proxy.

## Watch first
- [Rails 8 Deployment with Kamal](https://www.youtube.com/watch?v=yWSpjKErnco) (25 min) — Modern Rails deployment walkthrough with practical examples
- [Deploy a Rails App](https://www.youtube.com/watch?v=GKbx6s8_Qbo) (20 min) — Step-by-step Rails deployment tutorial for beginners

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand the full Rails deployment flow: server setup, code transfer, dependencies, database, app server, reverse proxy
- [ ] You know what Puma is and how it differs from PM2
- [ ] You understand why `rails db:migrate` must run on the server
- [ ] You have either deployed to a real server OR walked through the process locally step by step
- [ ] You can explain the difference between deploying a Node app and deploying a Rails app

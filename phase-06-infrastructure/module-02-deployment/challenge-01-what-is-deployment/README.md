# Challenge: What Is Deployment?

## What you'll learn
What it means to deploy an application, the steps involved, and the difference between development and production environments.

## Watch first
- [How to Deploy a Website](https://www.youtube.com/watch?v=_t-GjklnOgI) (8 min) — Simple overview of what deployment involves and the main approaches
- [DevOps CI/CD Explained in 100 Seconds](https://www.youtube.com/watch?v=scEDHsr3APg) (2 min) — Quick refresher on how CI/CD connects to deployment

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what "deploying an app" means in your own words
- [ ] You understand the difference between development and production environments
- [ ] You can list the basic steps to deploy a web application
- [ ] You know what a VPS (Virtual Private Server) is
- [ ] You can name at least 3 hosting platforms (e.g., DigitalOcean, Railway, Render, Heroku)

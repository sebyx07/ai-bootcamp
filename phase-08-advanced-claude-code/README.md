# Phase 07: Advanced Claude Code

You already know how to use Claude Code. Now you'll learn to use it like a power user.

This phase covers the techniques, configurations, and workflows that separate beginners from experts. You'll learn how to write prompts that get better results, configure CLAUDE.md files that make Claude smarter about your projects, use skills and hooks for automation, harness subagents and MCP servers for advanced workflows, and build an entire project AI-first from planning to shipping.

## Modules

1. **Effective Prompting** — One-shot vs conversation, giving context, reviewing before accepting, prompting full features
2. **CLAUDE.md Mastery** — Project conventions, per-directory rules, team configs, designing from scratch
3. **Skills and Hooks** — Built-in skills, custom slash commands, automation hooks
4. **Subagents and Tools** — What subagents are, parallel work, MCP servers, connecting external tools
5. **AI-First Project** — Plan, configure, build, and ship a project entirely with AI

## Prerequisites

- Completed Phases 01-06 (or equivalent comfort with Claude Code, terminal, Git, and basic web development)

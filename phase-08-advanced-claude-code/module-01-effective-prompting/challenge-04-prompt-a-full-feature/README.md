# Challenge: Prompt a Full Feature

## What you'll learn
How to go from a feature idea to a complete implementation using everything you've learned about prompting: context, conversation flow, and review.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — Watch how Peter Yang goes from spec to working feature using plan mode, CLAUDE.md, and iterative prompts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You planned a real feature by describing context and goals (not step-by-step instructions)
- [ ] You built it through a multi-turn conversation, reviewing each step
- [ ] The feature works and you tested it
- [ ] You can explain the prompting strategy you used and why

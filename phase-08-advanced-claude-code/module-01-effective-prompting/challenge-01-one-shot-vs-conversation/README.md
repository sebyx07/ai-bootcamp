# Challenge: One-Shot vs Conversation

## What you'll learn
The difference between giving Claude one big prompt versus building up a solution through back-and-forth conversation, and when each approach works best.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — Peter Yang shows the iterative spec-then-build workflow in Claude Code

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You built something small using one long prompt (one-shot)
- [ ] You built the same thing using a multi-turn conversation
- [ ] You can explain which approach gave better results and why
- [ ] You know when to use each approach

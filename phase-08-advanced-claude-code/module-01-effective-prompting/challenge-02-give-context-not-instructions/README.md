# Challenge: Give Context, Not Instructions

## What you'll learn
Why telling Claude about your project ("this is a Rails app with PostgreSQL") produces better results than giving step-by-step instructions ("first create a file called X, then write function Y").

## Watch first
- [The ULTIMATE Claude Code Tutorial](https://www.youtube.com/watch?v=3HVH2Iuplqo) (25 min) — Sabrina Ramonov walks through building a real project with Claude Code, demonstrating how context-driven prompts produce better results

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You prompted Claude with step-by-step instructions and saw the limitations
- [ ] You prompted Claude with rich context and a goal, and saw the improvement
- [ ] You can explain why context beats instructions for AI coding assistants
- [ ] You rewrote at least one old prompt from instruction-style to context-style

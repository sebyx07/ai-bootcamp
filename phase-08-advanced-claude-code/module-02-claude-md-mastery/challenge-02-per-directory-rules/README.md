# Challenge: Per-Directory Rules

## What you'll learn
How to create CLAUDE.md files in specific directories to give Claude different instructions for different parts of your project.

## Watch first
- [The ULTIMATE Claude Code Tutorial](https://www.youtube.com/watch?v=3HVH2Iuplqo) (25 min) — Sabrina Ramonov covers CLAUDE.md configuration as part of a full project setup

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have a root CLAUDE.md with project-wide conventions
- [ ] You have at least two directory-level CLAUDE.md files with specific rules
- [ ] Claude follows different rules in different directories
- [ ] You can explain when to use root vs directory-level CLAUDE.md files

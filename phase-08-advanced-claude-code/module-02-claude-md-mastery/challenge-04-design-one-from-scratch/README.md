# Challenge: Design One from Scratch

## What you'll learn
How to analyze any project and design a complete CLAUDE.md configuration from scratch, combining root conventions, per-directory rules, and team standards.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — See the full process of setting up CLAUDE.md for a new project

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You chose a real project (yours or open source) and analyzed its conventions
- [ ] You wrote a complete CLAUDE.md configuration (root + at least one directory-level file)
- [ ] You tested it by generating new code and verifying it matches the project style
- [ ] You iterated on the CLAUDE.md at least twice based on test results

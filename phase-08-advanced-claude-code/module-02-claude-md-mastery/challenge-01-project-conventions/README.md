# Challenge: Project Conventions

## What you'll learn
How to write a CLAUDE.md file that teaches Claude about your project's conventions, so it generates code that fits your codebase perfectly.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — Watch how Peter Yang creates a CLAUDE.md with best practices before writing any code

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a CLAUDE.md file in a project root
- [ ] The file describes the tech stack, coding style, and project conventions
- [ ] You asked Claude to generate code and it followed your conventions
- [ ] You can explain what belongs in a project CLAUDE.md

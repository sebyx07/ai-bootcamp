# Challenge: Ship It

## What you'll learn
How to take a finished project from your local machine to a live deployment that anyone can access.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — Includes shipping with GitHub and Vercel at the end of the build

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your project is in a GitHub repository with a clean README
- [ ] The project is deployed and accessible via a URL
- [ ] You wrote a brief retrospective: what worked, what you'd do differently
- [ ] You can share a link to your project with anyone

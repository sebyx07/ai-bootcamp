# Challenge: Plan the Project

## What you'll learn
How to plan a complete project using an AI-first approach: defining scope, choosing technologies, and creating a specification before writing any code.

## Watch first
- [Claude Code Beginner's Tutorial — Build a Movie App in 15 Minutes](https://www.youtube.com/watch?v=GepHGs_CZdk) (17 min) — Watch the spec-first, plan-before-code workflow in action

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You chose a small but real project to build (not a tutorial exercise)
- [ ] You created a project specification with Claude's help using plan mode
- [ ] The spec covers: what it does, who it's for, what technologies to use, and key features
- [ ] You have a prioritized list of features for an MVP

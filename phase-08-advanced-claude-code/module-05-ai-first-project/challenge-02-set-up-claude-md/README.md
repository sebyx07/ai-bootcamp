# Challenge: Set Up CLAUDE.md

## What you'll learn
How to configure Claude Code for your specific project by writing a comprehensive CLAUDE.md before writing any code.

## Watch first
- [Every Claude Code Concept Explained for Normal People](https://www.youtube.com/watch?v=fYX6hHC9FhQ) (20 min) — Understand how CLAUDE.md, memory, and project context work together

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a CLAUDE.md tailored to your project's tech stack and conventions
- [ ] You set up any per-directory CLAUDE.md files your project needs
- [ ] You created at least one custom slash command for your project workflow
- [ ] Claude generates code that follows your project's style on the first try

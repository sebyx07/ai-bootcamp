# Challenge: Parallel Work

## What you'll learn
How to structure prompts that let Claude work on multiple things at once using subagents, dramatically speeding up complex tasks.

## Watch first
- [Every Claude Code Concept Explained for Normal People](https://www.youtube.com/watch?v=fYX6hHC9FhQ) (20 min) — Explains how Claude Code's architecture enables parallel operations

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You gave Claude a task with multiple independent parts
- [ ] You observed Claude handling parts in parallel
- [ ] You compared parallel execution time to doing things one at a time
- [ ] You can identify tasks that benefit from parallel work

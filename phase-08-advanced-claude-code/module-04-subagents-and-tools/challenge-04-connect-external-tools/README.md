# Challenge: Connect External Tools

## What you'll learn
How to configure Claude Code to use external tools and MCP servers in a real project workflow.

## Watch first
- [Claude Code Essentials — freeCodeCamp](https://www.youtube.com/watch?v=GGSLMNmFKqk) (free course) — Comprehensive course covering MCP server integration, GitHub workflows, and external tool configuration

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You configured at least one external tool or MCP server in your Claude Code settings
- [ ] You used Claude to interact with an external service through that connection
- [ ] You can explain how to add and configure MCP servers
- [ ] You understand the security implications of connecting external tools

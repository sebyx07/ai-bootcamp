# Challenge: What Are Subagents

## What you'll learn
How Claude Code can spawn independent sub-agents to handle complex tasks, and why this matters for larger projects.

## Watch first
- [The ULTIMATE Claude Code Tutorial](https://www.youtube.com/watch?v=3HVH2Iuplqo) (25 min) — Covers subagents as part of a complete Claude Code workflow

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what a subagent is in your own words
- [ ] You understand when Claude spawns subagents automatically
- [ ] You triggered Claude to use a subagent on a real task
- [ ] You know the difference between the main agent and a subagent

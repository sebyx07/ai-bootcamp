# Challenge: Hooks Automation

## What you'll learn
How to set up Claude Code hooks that automatically run actions before or after certain events, creating quality gates and automated workflows.

## Watch first
- [The ULTIMATE Claude Code Tutorial](https://www.youtube.com/watch?v=3HVH2Iuplqo) (25 min) — Sabrina Ramonov demonstrates setting up quality gate hooks for automated checks

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand what hooks are and when they trigger
- [ ] You set up at least one hook in your project's settings
- [ ] Your hook runs automatically at the right time
- [ ] You can explain the difference between pre-tool and post-tool hooks

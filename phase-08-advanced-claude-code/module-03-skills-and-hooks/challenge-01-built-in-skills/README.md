# Challenge: Built-in Skills

## What you'll learn
What Claude Code skills are, how to discover built-in slash commands, and how to use them to work faster.

## Watch first
- [The ULTIMATE Claude Code Tutorial](https://www.youtube.com/watch?v=3HVH2Iuplqo) (25 min) — Covers skills, hooks, and slash commands as part of building a real project

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You explored built-in slash commands with /help
- [ ] You used at least three different slash commands (like /plan, /review, /clear)
- [ ] You can explain what skills are and how they differ from regular prompts
- [ ] You know how to discover new slash commands

# Challenge: Custom Slash Commands

## What you'll learn
How to create your own slash commands that automate repetitive workflows specific to your projects.

## Watch first
- [Claude Code Tutorial: Build a YouTube Research Agent using Slash Commands](https://www.youtube.com/watch?v=247Z3jdw_hs) (20 min) — Peter Yang builds a custom slash command from scratch for a real use case

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created at least one custom slash command in the .claude/commands/ directory
- [ ] Your command does something useful for a real project workflow
- [ ] You tested the command and it works as expected
- [ ] You can explain how custom slash commands are structured

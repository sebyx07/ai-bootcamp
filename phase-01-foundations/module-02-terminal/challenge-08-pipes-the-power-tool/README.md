# Challenge: Pipes — The Power Tool

## What you'll learn
How to chain commands together using the pipe (`|`) operator, turning simple commands into powerful combinations.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 180-210) — Covers pipes, redirection, and command chaining

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You used the pipe `|` to send one command's output to another command
- [ ] You used `sort` to sort a list
- [ ] You used `wc` to count lines in a file
- [ ] You combined at least 3 commands with pipes in one line

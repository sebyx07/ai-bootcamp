# Challenge: Copy, Move, Delete

## What you'll learn
How to copy files with `cp`, move/rename files with `mv`, and delete files with `rm`.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 70-90) — Covers cp, mv, rm, and rmdir

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You copied a file using `cp`
- [ ] You moved a file to a different folder using `mv`
- [ ] You renamed a file using `mv`
- [ ] You deleted a file using `rm` (carefully!)
- [ ] You know why `rm` is permanent and there's no trash can

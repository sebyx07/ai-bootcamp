# Challenge: Where Am I?

## What you'll learn
How to find your current location in the file system using `pwd`, and how computer folders are organized.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 10-20) — Covers pwd and the file system structure

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You ran `pwd` and understand what it shows
- [ ] You can explain what a "path" is (like /home/yourname/Documents)
- [ ] You know what the home directory is
- [ ] You know what the root directory (/) is

# Challenge: Read and Edit Files

## What you'll learn
How to view file contents with `cat`, `less`, and `head`, and how to write text into files.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 50-70) — Covers cat, less, head, tail, and writing to files

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You used `cat` to display a file's contents
- [ ] You used `echo "text" > file.txt` to write text into a file
- [ ] You used `echo "more text" >> file.txt` to add text to an existing file
- [ ] You used `head` or `tail` to see just part of a file

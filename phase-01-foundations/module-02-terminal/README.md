# Module 02: Terminal

The terminal is a text-based way to talk to your computer. Instead of clicking icons and dragging files, you type short commands. It feels strange at first, but it's faster and more powerful than clicking around.

Every developer uses the terminal daily. By the end of this module, you will too.

## Challenges

1. **What Is a Terminal?** — Open it and understand what you're looking at
2. **Where Am I?** — Learn your location in the file system
3. **Moving Around** — Navigate between folders
4. **Create Files and Folders** — Build things from the command line
5. **Read and Edit Files** — Look at file contents and make changes
6. **Copy, Move, Delete** — File management from the terminal
7. **Find Things** — Search for files and text
8. **Pipes: The Power Tool** — Chain commands together

# Challenge: Moving Around

## What you'll learn
How to navigate between folders using `cd`, and the difference between absolute and relative paths.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 20-35) — Covers cd, relative paths, and absolute paths

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You used `cd` to move into a folder
- [ ] You used `cd ..` to move back up one level
- [ ] You used `cd ~` to go home
- [ ] You can explain the difference between a relative and absolute path

# Challenge: Create Files and Folders

## What you'll learn
How to create new files with `touch` and new folders with `mkdir` from the command line.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 35-50) — Covers touch, mkdir, and file creation

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a file using `touch`
- [ ] You created a folder using `mkdir`
- [ ] You created a nested folder structure with `mkdir -p`
- [ ] You built a mini project folder with at least 3 files in organized subfolders

# Challenge: Find Things

## What you'll learn
How to search for files with `find` and search inside files with `grep`.

## Watch first
- [The 50 Most Popular Linux & Terminal Commands](https://www.youtube.com/watch?v=ZtqBQ68cfJc) (watch minutes 150-180) — Covers find and grep in detail

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You used `find` to locate a file by name
- [ ] You used `grep` to search for text inside a file
- [ ] You searched for a word across multiple files at once
- [ ] You can explain when to use `find` vs `grep`

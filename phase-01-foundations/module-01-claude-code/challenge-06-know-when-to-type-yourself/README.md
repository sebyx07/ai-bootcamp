# Challenge: Know When to Type Yourself

## What you'll learn
When to use Claude and when it's faster or better to just type something yourself.

## Watch first
- [Claude Code in Action — Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action) (20 min) — Notice which tasks are quick to do yourself vs. which benefit from AI help

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You completed a task by typing a command yourself instead of asking Claude
- [ ] You can name 3 things that are faster to do yourself
- [ ] You can name 3 things where Claude genuinely helps
- [ ] You created a file by hand and verified it exists

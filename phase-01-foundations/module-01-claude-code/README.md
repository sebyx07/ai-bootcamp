# Module 01: Claude Code

This module teaches you how to use Claude Code — your AI coding assistant and teacher for this entire bootcamp.

Claude Code is a tool that lives in your terminal. You type messages to it, and it helps you write code, understand concepts, and fix problems. But like any tool, it works better when you know how to use it well.

## Challenges

1. **Talk to Claude** — Open Claude Code and have your first conversation
2. **Ask Claude to Do Things** — Learn to give Claude tasks, not just questions
3. **Short Prompts Win** — The most important skill: saying less to get more
4. **Read What Claude Did** — Understanding Claude's output before moving on
5. **Fix Something Together** — Collaborate with Claude to debug a problem
6. **Know When to Type Yourself** — Not everything needs AI help

## Why this module matters

Every other module in this bootcamp uses Claude Code as your teacher. Learning to communicate with it effectively is like learning to ask good questions in a classroom — it makes everything else easier.

## Time estimate

2-3 hours

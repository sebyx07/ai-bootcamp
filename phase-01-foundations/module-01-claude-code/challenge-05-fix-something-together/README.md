# Challenge: Fix Something Together

## What you'll learn
How to work with Claude to find and fix a problem — the basics of debugging as a team.

## Watch first
- [Claude Code in Action — Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action) (20 min) — See how Claude helps identify and fix issues in code

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Claude created a file with a deliberate mistake in it
- [ ] You spotted the mistake (or Claude helped you spot it)
- [ ] You told Claude what to fix in plain English
- [ ] The file is now correct and you verified it

# Challenge: Short Prompts Win

## What you'll learn
The single most important prompting skill: keeping your requests short and direct instead of writing essays.

## Watch first
- [Claude Code in Action — Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action) (20 min) — Pay attention to how the examples use short, clear prompts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You rewrote a long prompt into a short one and saw it work just as well
- [ ] You completed 3 tasks using prompts of 10 words or fewer
- [ ] You can explain why short prompts often work better than long ones
- [ ] You have a feel for when to add detail and when to keep it brief

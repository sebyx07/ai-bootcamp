# Challenge: Talk to Claude

## What you'll learn
How to open Claude Code and have your first conversation with an AI coding assistant.

## Watch first
- [Claude Code 101 — Anthropic Academy](https://anthropic.skilljar.com/claude-code-101) (20 min) — Official free course covering what Claude Code is and how to start using it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You opened Claude Code in your terminal
- [ ] You asked Claude at least 3 different questions
- [ ] You asked Claude to explain what Claude Code is
- [ ] You know how to exit Claude Code (type /exit or press Ctrl+C)

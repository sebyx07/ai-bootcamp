# Challenge: Ask Claude to Do Things

## What you'll learn
The difference between asking Claude questions and asking Claude to perform tasks on your computer.

## Watch first
- [Claude Code in Action — Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action) (20 min) — See Claude Code performing real tasks: creating files, editing code, and running commands

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You asked Claude to create a new file on your computer
- [ ] You asked Claude to write something inside that file
- [ ] You asked Claude to show you the contents of the file
- [ ] You understand the difference between asking a question and giving a task

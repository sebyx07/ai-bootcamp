# Challenge: Read What Claude Did

## What you'll learn
How to understand Claude's output — the commands it ran, the files it changed, and why it matters to read before moving on.

## Watch first
- [Claude Code in Action — Anthropic Academy](https://anthropic.skilljar.com/claude-code-in-action) (20 min) — Watch how the instructor reviews what Claude did after each step

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You asked Claude to do a multi-step task and read each step it took
- [ ] You can point to a command Claude ran and explain what it did
- [ ] You caught at least one thing Claude did that you didn't expect
- [ ] You asked Claude "what did you just do?" and understood the answer

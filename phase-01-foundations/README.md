# Phase 01: Foundations

Welcome to the starting line. This phase teaches you the three tools every developer uses daily: an AI coding assistant (Claude Code), the terminal, and Git version control.

You don't need any prior experience. You don't need to know what "code" is yet. By the end of this phase, you'll be comfortable talking to your computer through text, navigating the file system, and tracking every change you make.

## What you'll learn

- **Module 01 — Claude Code**: How to use your AI mentor effectively. Asking good questions, reading what it does, and knowing when to type yourself.
- **Module 02 — Terminal**: The text-based interface to your computer. Moving around, creating files, reading them, and combining commands.
- **Module 03 — Git**: Version control from scratch. Saving your work, branching, merging, and collaborating on GitHub.
- **Module 04 — GitHub**: The GitHub platform. Issues, pull requests, code reviews, CI/CD with GitHub Actions, and branch protection.

## How it works

Each module has challenges. For every challenge:

1. Read the short README
2. Watch the linked video (5-25 minutes)
3. Open Claude Code and say "I'm ready for the challenge"
4. Claude will teach you interactively
5. Complete the "done when" checklist

## Estimated time

- Module 01: 2-3 hours
- Module 02: 4-6 hours
- Module 03: 4-6 hours
- Module 04: 3-5 hours

Total: roughly 13-20 hours at a relaxed pace. Take breaks. Let things sink in.

## Prerequisites

- A computer with Claude Code installed
- A terminal (you already have one, we'll find it)
- Curiosity and patience

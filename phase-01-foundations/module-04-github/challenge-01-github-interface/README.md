# Challenge: The GitHub Interface

## What you'll learn
How to navigate a GitHub repository page and understand what each tab and section shows you.

## Watch first
- [Git and GitHub for Beginners — Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) (69 min) — freeCodeCamp overview of Git and GitHub; watch the GitHub sections starting around 30 minutes in
- [Git and GitHub for Beginners Tutorial](https://www.youtube.com/watch?v=tRZGeaHPoaw) (46 min) — Kevin Stratvert walks through the GitHub interface step by step; skip to the GitHub.com portions if you already know Git basics

## Open Claude Code and say:
> "I'm ready for the GitHub interface challenge"

## You're done when
- [ ] You can name at least 5 tabs on a GitHub repo page and explain what each one shows
- [ ] You have explored the commits tab and can find who changed what and when
- [ ] You have looked at the branches page and understand how it relates to `git branch`
- [ ] You have found the repo settings page and know where it lives
- [ ] You can explain the difference between the "Code" tab and the "Issues" tab in your own words

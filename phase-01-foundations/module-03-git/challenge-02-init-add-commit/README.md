# Challenge: Init, Add, Commit

## What you'll learn
The three core Git commands: creating a repository, staging changes, and saving a snapshot.

## Watch first
- [Git & GitHub Crash Course 2025](https://www.youtube.com/watch?v=vA5TTz6BXhY) (watch first 15 min) — Traversy Media walks through init, add, and commit step by step
- [Git and GitHub for Beginners — Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) (watch minutes 10-25) — freeCodeCamp covers the same commands with clear examples

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a new Git repository with `git init`
- [ ] You added files to the staging area with `git add`
- [ ] You created your first commit with `git commit`
- [ ] You ran `git status` and understand what it shows
- [ ] You made at least 3 commits with meaningful messages

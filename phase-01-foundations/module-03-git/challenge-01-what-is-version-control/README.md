# Challenge: What Is Version Control?

## What you'll learn
Why version control exists, what problem it solves, and how Git fits into the picture.

## Watch first
- [Git Explained in 100 Seconds](https://www.youtube.com/watch?v=hwP7WQkmECE) (2 min) — Fireship's fast, visual explanation of what Git is
- [Git and GitHub for Beginners — Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) (watch first 10 min) — freeCodeCamp's beginner-friendly intro to version control concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what version control is in your own words
- [ ] You can name 3 problems that Git solves
- [ ] You understand the difference between Git and GitHub
- [ ] You have Git installed on your computer

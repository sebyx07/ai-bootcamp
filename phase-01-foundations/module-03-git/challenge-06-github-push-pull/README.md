# Challenge: GitHub — Push and Pull

## What you'll learn
How to create a GitHub repository and sync your local code with the cloud using `git push` and `git pull`.

## Watch first
- [Git & GitHub Crash Course 2025](https://www.youtube.com/watch?v=vA5TTz6BXhY) (watch minutes 20-35) — Traversy Media covers connecting to GitHub and pushing code
- [Git and GitHub for Beginners — Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) (watch minutes 25-40) — freeCodeCamp covers remotes, push, and pull

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a repository on GitHub.com
- [ ] You connected your local repo to GitHub with `git remote add`
- [ ] You pushed your code to GitHub with `git push`
- [ ] You can see your code on GitHub.com in a browser
- [ ] You understand what `origin` means

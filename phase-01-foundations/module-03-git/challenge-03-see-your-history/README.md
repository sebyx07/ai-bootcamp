# Challenge: See Your History

## What you'll learn
How to view your commit history with `git log` and see what changed with `git diff`.

## Watch first
- [Git & GitHub Crash Course 2025](https://www.youtube.com/watch?v=vA5TTz6BXhY) (watch minutes 15-25) — Traversy Media covers git log and reviewing changes
- [Git Explained in 100 Seconds](https://www.youtube.com/watch?v=hwP7WQkmECE) (2 min) — Quick refresher on how Git tracks history

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You ran `git log` and can read the output
- [ ] You used `git log --oneline` for a compact view
- [ ] You used `git diff` to see what changed in a file
- [ ] You can find a specific commit in your history

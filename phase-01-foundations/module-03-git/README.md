# Module 03: Git

Git is a version control system. It tracks every change you make to your files, lets you undo mistakes, and makes it possible for multiple people to work on the same project without stepping on each other's toes.

Think of it as unlimited undo, save points, and parallel universes for your code — all at once.

## Challenges

1. **What Is Version Control?** — Understand why Git exists
2. **Init, Add, Commit** — Create your first repository and save your first snapshot
3. **See Your History** — Look at what you've done over time
4. **Branches** — Work on ideas without breaking what already works
5. **Merge and Conflicts** — Bring branches together and handle disagreements
6. **GitHub: Push and Pull** — Put your code in the cloud
7. **Pull Requests** — The standard way teams review code
8. **Daily Git Workflow** — The commands you'll use every single day

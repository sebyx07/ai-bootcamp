# Challenge: Daily Git Workflow

## What you'll learn
The complete daily workflow that professional developers use with Git — the commands you'll run every single day.

## Watch first
- [Git & GitHub Crash Course 2025](https://www.youtube.com/watch?v=vA5TTz6BXhY) (watch full video, 49 min) — Traversy Media's complete walkthrough ties everything together
- [Git Explained in 100 Seconds](https://www.youtube.com/watch?v=hwP7WQkmECE) (2 min) — Rewatch Fireship's overview now that you understand every concept mentioned

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You completed a full cycle: pull, branch, code, commit, push, PR, merge
- [ ] You can do the daily workflow without looking at notes
- [ ] You used `git stash` to save work temporarily
- [ ] You wrote a cheat sheet of your most-used Git commands

# Challenge: Branches

## What you'll learn
How to create branches to work on new ideas without affecting your main code.

## Watch first
- [Git & GitHub Crash Course 2025](https://www.youtube.com/watch?v=vA5TTz6BXhY) (watch minutes 25-35) — Traversy Media explains branches with visual examples
- [Git and GitHub for Beginners — Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) (watch minutes 30-45) — freeCodeCamp covers branching and switching between branches

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You created a new branch with `git branch` or `git checkout -b`
- [ ] You switched between branches
- [ ] You made commits on a branch that don't appear on main
- [ ] You can explain why branches are useful

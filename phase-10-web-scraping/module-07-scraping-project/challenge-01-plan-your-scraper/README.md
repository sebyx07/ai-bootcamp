# Challenge: Plan Your Scraper

## What you'll learn
How to plan a real scraping project — picking an ethical target, deciding what data to extract, and choosing the right tool for the job.

## Watch first
- [Web Scraping Best Practices — John Watson Rooney](https://www.youtube.com/watch?v=Lo62LRFadYM) (12 min) — practical advice on planning and executing scraping projects responsibly

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You picked a target site with ethical considerations in mind
- [ ] You identified the specific data points you want to extract
- [ ] You chose your tool (Playwright or Ferrum) and can explain why
- [ ] You have a written plan documenting target, data points, and approach

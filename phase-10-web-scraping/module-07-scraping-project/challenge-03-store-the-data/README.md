# Challenge: Store the Data

## What you'll learn
How to save scraped data into structured formats like JSON, CSV, or a database so it can be queried and used later.

## No video needed
This is a hands-on building challenge — you will extend the scraper you built in challenge-02.

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Scraped data is saved to a file (JSON or CSV) or a database
- [ ] The data is structured and organized (not just raw text)
- [ ] You can query or filter the stored data
- [ ] You understand when to use a file vs a database for scraped data

# Challenge: Build the Scraper

## What you'll learn
How to turn a scraping plan into working code by choosing the right tool and building the scraper step by step.

## No video needed
This is a hands-on building challenge — you and Claude Code will write the scraper together.

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You chose the right tool (Playwright, Ferrum, or Cheerio) for your target site
- [ ] The scraper runs without errors
- [ ] It extracts the data you planned in challenge-01
- [ ] You can explain why you chose that tool over the alternatives

# Phase 10: Web Scraping & Reverse Engineering

You've built frontends and backends. Now you'll learn how to extract data from existing websites — ethically and effectively.

This phase covers browser automation with Playwright (JavaScript) and Ferrum (Ruby), lightweight scraping with Cheerio, traffic interception with mitmproxy, and reverse engineering techniques to understand how websites and apps work under the hood. These are real-world skills used in data collection, testing, debugging, and competitive analysis.

## Modules

1. **Scraping Fundamentals** — What web scraping is, how it works, ethics and legality
2. **Playwright** — Browser automation with JavaScript using Playwright
3. **Ferrum** — Browser automation with Ruby using Ferrum (headless Chrome)
4. **JS Scraping Without a Browser** — Cheerio, Puppeteer, and lightweight scraping in JavaScript
5. **MITM & Traffic Interception** — Using mitmproxy to intercept, inspect, and modify HTTP traffic
6. **Reverse Engineering** — Using DevTools to find hidden APIs and understand how sites work
7. **Scraping Project** — Build a real scraper from scratch

## Prerequisites

- Completed Phase 03 (Frontend — HTML, CSS, JavaScript)
- Completed Phase 04 (Backend — HTTP, Node/Express, Ruby basics)
- Comfortable reading HTML and using the terminal

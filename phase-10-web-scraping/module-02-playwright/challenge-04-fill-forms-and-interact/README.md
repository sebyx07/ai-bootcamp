# Challenge: Fill Forms and Interact

## What you'll learn
How to use Playwright to fill out forms, select dropdown options, check checkboxes, and submit forms — essential for automating web interactions.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Covers form interactions and input handling

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your script fills out text inputs on a form
- [ ] The script selects a dropdown option
- [ ] The script checks a checkbox
- [ ] The script submits the form and verifies success

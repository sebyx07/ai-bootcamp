# Challenge: What Is Playwright?

## What you'll learn
What Playwright is, how browser automation works, and the difference between headless and headed browsers.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Overview of Playwright and browser automation concepts

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what Playwright does in one sentence
- [ ] You understand headless vs headed browsers
- [ ] You know why Playwright is better than just fetching HTML with curl
- [ ] You can name at least two use cases for browser automation

# Challenge: Screenshots and PDFs

## What you'll learn
How to capture full-page screenshots, generate PDFs from web pages, and screenshot specific elements using Playwright.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Covers screenshot and PDF generation

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your script takes a full-page screenshot (not just the viewport)
- [ ] Your script generates a PDF of a web page
- [ ] Your script captures a screenshot of a specific element on the page
- [ ] You have saved files for each: screenshot, full-page screenshot, element screenshot, and PDF

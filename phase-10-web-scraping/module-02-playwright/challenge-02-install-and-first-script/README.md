# Challenge: Install Playwright and First Script

## What you'll learn
How to install Playwright and write your first script that opens a browser, navigates to a URL, and takes a screenshot.

## Watch first
- [Getting Started with Playwright — Playwright official](https://www.youtube.com/watch?v=Xz6lhEzgI5I) (10 min) — Official getting started walkthrough

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Playwright is installed and working
- [ ] You have a script that opens a browser and goes to a website
- [ ] The script takes a screenshot and saves it as an image file
- [ ] You can run the script and see the screenshot output

# Challenge: Scrape a Dynamic Page

## What you'll learn
How to combine all your Playwright skills to scrape structured data from a JavaScript-rendered page and save it as clean JSON.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Full walkthrough of scraping with Playwright

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your script navigates to a JavaScript-rendered page
- [ ] The script waits for dynamic content to load
- [ ] The script extracts structured data (titles, prices, descriptions, etc.)
- [ ] The extracted data is saved as a clean JSON file
- [ ] The JSON file contains an array of objects with consistent keys

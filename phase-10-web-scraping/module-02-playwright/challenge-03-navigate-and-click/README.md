# Challenge: Navigate and Click

## What you'll learn
How to use Playwright to navigate between pages, click buttons, and follow links — the building blocks of browser automation.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Covers navigation and interaction basics

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your script navigates to a website and clicks on links
- [ ] The script follows links to navigate through multiple pages
- [ ] You can extract text from the pages you visit
- [ ] You understand how Playwright locates elements to click on

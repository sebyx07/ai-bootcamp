# Challenge: Wait for Elements

## What you'll learn
How to handle dynamic content that loads asynchronously — using waitForSelector, waitForNavigation, and timeouts to make your scripts reliable.

## Watch first
- [Playwright Tutorial for Beginners — LambdaTest](https://www.youtube.com/watch?v=wawbt1cATsk) (15 min) — Covers waiting strategies and handling async content

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You understand why waiting is necessary for dynamic pages
- [ ] Your script waits for an element to appear before interacting with it
- [ ] You can handle pages where content loads after the initial page load
- [ ] You know the difference between different waiting strategies

# Challenge: Understand Request Headers

## What you'll learn
Why HTTP headers matter — how sites use User-Agent, cookies, authorization, content-type, and referer to decide whether to serve you data or block you.

## Watch first
- [Chrome DevTools Network Tab — Google Chrome Developers](https://www.youtube.com/watch?v=e1gAyQuIFQo) (10 min) — focus on the Headers section of requests this time

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can identify the key headers in a request (User-Agent, Cookie, Authorization, Content-Type, Referer)
- [ ] You can explain why a site might check these headers
- [ ] You understand which headers you'd need to include when replaying a request

# Challenge: Replay API Calls

## What you'll learn
How to take a discovered API call from the browser and replay it using curl or a script — the key skill that turns observation into automation.

## Watch first
- [Chrome DevTools Network Tab — Google Chrome Developers](https://www.youtube.com/watch?v=e1gAyQuIFQo) (10 min) — pay attention to "Copy as cURL" feature

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can copy an API call from the Network tab as a curl command
- [ ] You successfully replay an API call outside the browser and get data back
- [ ] You can modify parameters (e.g., search term, page number) in the replayed call

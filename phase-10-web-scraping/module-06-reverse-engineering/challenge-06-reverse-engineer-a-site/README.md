# Challenge: Reverse Engineer a Site

## What you'll learn
How to combine all reverse engineering skills into a full workflow — discover a site's hidden API, understand its request format, replay it, and extract data programmatically.

## Watch first
- [Reverse Engineering Web Apps — LiveOverflow](https://www.youtube.com/watch?v=2dvRKS8XUAE) (15 min) — rewatch with fresh eyes now that you have hands-on experience

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You picked a real website and found its hidden API
- [ ] You understand the API's request format (URL pattern, required headers, parameters)
- [ ] You replayed the API call and modified parameters to get different data
- [ ] You wrote a script that extracts data from the API programmatically

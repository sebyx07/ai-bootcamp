# Challenge: What Is Reverse Engineering?

## What you'll learn
How to understand websites from the outside — finding patterns, discovering hidden APIs, and figuring out how things work under the hood.

## Watch first
- [Reverse Engineering Web Apps — LiveOverflow](https://www.youtube.com/watch?v=2dvRKS8XUAE) (15 min) — explains the mindset and techniques behind reverse engineering websites

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what reverse engineering means in a web context
- [ ] You can describe why reverse engineering is useful for scraping
- [ ] You understand the difference between scraping HTML and finding hidden APIs

# Challenge: Browser DevTools Network Tab

## What you'll learn
How to use the Network tab in Chrome DevTools to inspect XHR requests, filter traffic, and read API request/response data.

## Watch first
- [Chrome DevTools Network Tab — Google Chrome Developers](https://www.youtube.com/watch?v=e1gAyQuIFQo) (10 min) — official walkthrough of the Network tab features

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can open the Network tab and filter for XHR/Fetch requests
- [ ] You can find API calls happening behind the scenes on a real website
- [ ] You can inspect the request URL, headers, and response body of an API call

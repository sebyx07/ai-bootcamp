# Challenge: Find Hidden APIs

## What you'll learn
How to discover undocumented internal APIs that websites use to load data — endpoints that aren't public but are visible in the Network tab.

## Watch first
- [Chrome DevTools Network Tab — Google Chrome Developers](https://www.youtube.com/watch?v=e1gAyQuIFQo) (10 min) — review the Network tab skills you'll use here

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You found at least 2 hidden API endpoints on a real website
- [ ] You can describe what data each endpoint returns
- [ ] You understand how user actions (search, scroll, click) trigger API calls

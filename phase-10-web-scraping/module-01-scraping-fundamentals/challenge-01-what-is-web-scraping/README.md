# Challenge: What Is Web Scraping

## What you'll learn
What web scraping is, why people do it, and real-world examples like price monitoring, research, and data journalism.

## Watch first
- [Web Scraping explained — Fireship](https://www.youtube.com/watch?v=4jk9H5AB4lM) (5 min) — Quick overview of what web scraping is and how it works

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what web scraping is in your own words
- [ ] You know the difference between scraping and using an API
- [ ] You can name at least 3 real-world uses of web scraping
- [ ] You understand when scraping is the right tool vs other options

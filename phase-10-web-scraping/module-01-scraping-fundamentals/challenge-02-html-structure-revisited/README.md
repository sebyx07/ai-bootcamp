# Challenge: HTML Structure Revisited

## What you'll learn
How to look at HTML from a scraper's perspective — using CSS selectors and DevTools to target the exact elements you want to extract.

## Watch first
- [CSS Selectors crash course — Web Dev Simplified](https://www.youtube.com/watch?v=l1mER1bV0N0) (20 min) — Deep dive into CSS selectors, which are the same selectors you use to target elements when scraping

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can open DevTools and inspect any element on a webpage
- [ ] You can write a CSS selector to target a specific element
- [ ] You used DevTools to find selectors for at least 3 different elements on a real website
- [ ] You understand how classes, IDs, and nested selectors work for targeting data

# Challenge: HTTP Requests and Responses

## What you'll learn
How HTTP works from a scraper's perspective — making GET requests, reading response codes, and understanding headers and cookies.

## Watch first
- [HTTP Crash Course — Traversy Media](https://www.youtube.com/watch?v=iYM2zFP3Zn0) (35 min) — Comprehensive walkthrough of HTTP requests, responses, status codes, and headers

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You used curl to fetch a webpage and saw the raw HTML response
- [ ] You inspected HTTP headers using curl -I
- [ ] You can explain what status codes 200, 403, and 404 mean in a scraping context
- [ ] You understand why headers matter when scraping

# Challenge: Ethics and Legality

## What you'll learn
The rules of responsible scraping — robots.txt, rate limiting, Terms of Service, and how to be a "polite" scraper that does not cause problems.

## Watch first
- [Is Web Scraping Legal? — John Watson Rooney](https://www.youtube.com/watch?v=tcMBh9bqxuI) (10 min) — Breaks down the legal landscape of web scraping and what you need to know

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You checked robots.txt for 3 popular websites and can explain what you found
- [ ] You understand what robots.txt is and what it controls
- [ ] You know why adding delays between requests matters
- [ ] You can list 3 rules for being a responsible scraper

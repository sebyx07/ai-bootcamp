# Challenge: Capture Mobile App Traffic

## What you'll learn
How to configure a phone or emulator to route traffic through mitmproxy, revealing the hidden API calls that mobile apps make behind the scenes.

## Watch first
- [Getting Started with mitmproxy — NeuralNine](https://www.youtube.com/watch?v=7bL-eNsGMcM) (15 min) — Review the mobile proxy configuration sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] A phone or emulator is configured to use mitmproxy as its proxy
- [ ] The mitmproxy CA certificate is installed on the mobile device
- [ ] You can see API traffic from a mobile app flowing through mitmproxy
- [ ] You can identify at least one undocumented API endpoint from an app

# Challenge: Inspect and Modify Requests

## What you'll learn
How to inspect full request/response details in mitmproxy and modify requests on the fly to change what gets sent to a server.

## Watch first
- [Getting Started with mitmproxy — NeuralNine](https://www.youtube.com/watch?v=7bL-eNsGMcM) (15 min) — Focus on the request editing and interception sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can inspect headers, body, and cookies of any captured request
- [ ] You can intercept a request and modify it before it reaches the server
- [ ] You can see how a modified request produces a different response
- [ ] You understand when and why you'd modify requests during development

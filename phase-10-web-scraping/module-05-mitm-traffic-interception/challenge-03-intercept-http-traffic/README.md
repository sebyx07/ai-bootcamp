# Challenge: Intercept HTTP Traffic

## What you'll learn
How to use mitmproxy to watch HTTP/HTTPS requests in real time, filter traffic, and search for specific requests.

## Watch first
- [Getting Started with mitmproxy — NeuralNine](https://www.youtube.com/watch?v=7bL-eNsGMcM) (15 min) — Review the traffic inspection portions if needed

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can browse a website and see all requests flowing through mitmproxy
- [ ] You can filter traffic to show only specific domains or paths
- [ ] You can search for a specific request in the traffic flow
- [ ] You can explain what the different parts of a captured request mean

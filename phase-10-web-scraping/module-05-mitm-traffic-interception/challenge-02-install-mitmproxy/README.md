# Challenge: Install mitmproxy

## What you'll learn
How to install mitmproxy, configure your browser to route traffic through it, and set up HTTPS interception with a CA certificate.

## Watch first
- [Getting Started with mitmproxy — NeuralNine](https://www.youtube.com/watch?v=7bL-eNsGMcM) (15 min) — Walkthrough of installing and configuring mitmproxy from scratch

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] mitmproxy is installed and runs without errors
- [ ] Your browser is configured to use mitmproxy as its proxy
- [ ] The mitmproxy CA certificate is installed for HTTPS interception
- [ ] You can see browser traffic flowing through mitmproxy

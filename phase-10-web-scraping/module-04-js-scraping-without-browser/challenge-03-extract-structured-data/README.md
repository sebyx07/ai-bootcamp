# Challenge: Extract Structured Data

## What you'll learn
How to extract multiple items from a page and structure them as clean JSON using Cheerio's iteration methods.

## Watch first
- [Web Scraping with Node.js and Cheerio — Traversy Media](https://www.youtube.com/watch?v=PnHqOGDnEBk) (30 min) — focus on extracting lists and multiple elements

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Script extracts multiple items from a page (not just one element)
- [ ] Uses `.each()`, `.map()`, `.text()`, and `.attr()` methods
- [ ] Outputs a clean JSON array of structured data
- [ ] Each item in the array has consistent keys (e.g. title, url, description)

# Challenge: What is Cheerio?

## What you'll learn
What Cheerio is, how it parses HTML using jQuery-like syntax without a browser, and when to use it instead of Playwright.

## Watch first
- [Web Scraping with Node.js and Cheerio — Traversy Media](https://www.youtube.com/watch?v=PnHqOGDnEBk) (30 min) — covers Cheerio basics, selecting elements, and extracting data from static HTML

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what Cheerio is and how it works
- [ ] You know when to use Cheerio vs Playwright
- [ ] You can describe what Cheerio can't do (and why)

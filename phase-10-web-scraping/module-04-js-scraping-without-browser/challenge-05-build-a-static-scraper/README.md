# Challenge: Build a Static Scraper

## What you'll learn
How to build a complete scraper using Cheerio and fetch that extracts structured data from a static site and saves it to a JSON file.

## Watch first
- [Web Scraping with Node.js and Cheerio — Traversy Media](https://www.youtube.com/watch?v=PnHqOGDnEBk) (30 min) — review the full workflow from fetch to structured output

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Scraper fetches HTML from a real static website
- [ ] Extracts structured data using Cheerio selectors
- [ ] Saves clean, formatted JSON to a file
- [ ] Script runs end-to-end without errors
- [ ] Output JSON has consistent structure with meaningful fields

# Challenge: Puppeteer vs Playwright

## What you'll learn
How Puppeteer and Playwright compare as browser automation tools, and when to choose each one.

## Watch first
- [Web Scraping with Node.js and Cheerio — Traversy Media](https://www.youtube.com/watch?v=PnHqOGDnEBk) (30 min) — review the landscape of scraping tools discussed

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain the scraping tool landscape: Cheerio (static), Puppeteer (Google), Playwright (Microsoft)
- [ ] You've tried the same scraping task with both Puppeteer and Playwright
- [ ] You can describe the tradeoffs between all three tools
- [ ] You know when to reach for each tool

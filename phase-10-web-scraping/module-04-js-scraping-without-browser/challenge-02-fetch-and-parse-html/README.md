# Challenge: Fetch and Parse HTML

## What you'll learn
How to download a web page's HTML with fetch and parse it with Cheerio to extract specific content.

## Watch first
- [Web Scraping with Node.js and Cheerio — Traversy Media](https://www.youtube.com/watch?v=PnHqOGDnEBk) (30 min) — focus on the fetch + Cheerio workflow sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Script uses fetch (or axios) to download a page's HTML
- [ ] Script parses the HTML with Cheerio
- [ ] Script selects specific elements using CSS selectors
- [ ] Script prints extracted text to the console

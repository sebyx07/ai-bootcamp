# Challenge: Scrape a Real Site

## What you'll learn
How to combine all your Ferrum skills to scrape structured data from a real website and save it as clean JSON.

## Watch first
- [Web Scraping with Ruby — Deanin](https://www.youtube.com/watch?v=yKPL5Bkuqoo) (20 min) — review the full project sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your Ruby script scrapes structured data from a real website
- [ ] The scraped data is saved as a clean JSON file
- [ ] Your script handles errors and timeouts gracefully
- [ ] You can explain when you'd choose Ferrum over Playwright (and vice versa)
- [ ] You've compared your Ferrum and Playwright scraping experiences

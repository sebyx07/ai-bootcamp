# Challenge: Navigate and Extract Data

## What you'll learn
How to use Ferrum's CSS selector methods to find elements on a page and extract text, links, and images.

## Watch first
- [Web Scraping with Ruby — Deanin](https://www.youtube.com/watch?v=yKPL5Bkuqoo) (20 min) — focus on the data extraction sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can use `browser.at_css` to find a single element
- [ ] You can use `browser.css` to find multiple elements
- [ ] Your script extracts text content from elements
- [ ] Your script extracts links (href attributes) from a page
- [ ] Your script prints the extracted data to the terminal

# Challenge: Install Ferrum and First Script

## What you'll learn
How to install the Ferrum gem, launch a headless browser from Ruby, navigate to a URL, and take a screenshot.

## Watch first
- [Web Scraping with Ruby — Deanin](https://www.youtube.com/watch?v=yKPL5Bkuqoo) (20 min) — watch the setup and first script sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Ferrum gem is installed
- [ ] You have a Ruby script that opens a headless browser
- [ ] The script navigates to a URL
- [ ] The script saves a screenshot as a PNG file
- [ ] You can explain what each line of the script does

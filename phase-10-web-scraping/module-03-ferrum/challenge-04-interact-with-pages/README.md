# Challenge: Interact with Pages

## What you'll learn
How to use Ferrum to click buttons, fill in forms, select dropdown options, and interact with page elements like a real user.

## Watch first
- [Web Scraping with Ruby — Deanin](https://www.youtube.com/watch?v=yKPL5Bkuqoo) (20 min) — review the form interaction sections

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Your script can fill in a text input field
- [ ] Your script can click a button
- [ ] Your script can select a dropdown option
- [ ] You tested the interactions on a practice site
- [ ] You can explain the difference between finding an element and interacting with it

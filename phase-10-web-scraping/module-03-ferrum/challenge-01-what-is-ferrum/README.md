# Challenge: What Is Ferrum?

## What you'll learn
What Ferrum is, how it controls Chrome from Ruby using the Chrome DevTools Protocol, and how it compares to Playwright.

## Watch first
- [Web Scraping with Ruby — Deanin](https://www.youtube.com/watch?v=yKPL5Bkuqoo) (20 min) — overview of web scraping in Ruby and the tools available

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what Ferrum is in one sentence
- [ ] You know what CDP (Chrome DevTools Protocol) means
- [ ] You can describe how Ferrum compares to Playwright
- [ ] You understand when you'd pick Ferrum over Playwright (and vice versa)

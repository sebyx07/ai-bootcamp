# Capstone Option C: Your Own Idea

## The Project

Build whatever you want. Seriously. The only requirements are scope and skill demonstration.

## Requirements

### Must Have
- Uses skills from at least three phases of the bootcamp (e.g., frontend + backend + deployment, or terminal tools + testing + Git workflows)
- Has a clear purpose — it solves a problem or serves a need
- Is scoped to be completable in 1-2 weeks
- Is deployed and accessible
- Has a CLAUDE.md that you wrote yourself
- Has a README that explains what it is and how to use it

### The Bar
Your project should demonstrate that you can:
- Plan a project before building it
- Build it using AI-first workflows
- Write clean, tested code
- Deploy it to the internet
- Explain what you built and why

## Ideas to Spark Inspiration

- A CLI tool that does something useful (file organizer, project scaffolder, data analyzer)
- A browser extension
- A mobile-friendly web app for a personal need
- An automation tool that connects multiple services
- A data visualization dashboard
- A tool for your hobby (fitness tracker, music catalog, recipe scaler)
- A game
- A developer tool that you wish existed

## How to Scope

The biggest risk is building something too ambitious. Use this test:

1. Describe the project in one sentence
2. List the core features (no more than five)
3. Cut the list in half
4. That is your MVP

If your MVP takes more than one week, cut it again.

# Challenge: Dev Environment in Docker

## What you'll learn
How to use Docker as your development environment, so your entire project setup is portable and reproducible.

## Watch first
- [How To Use Docker To Make Local Development Easier](https://www.youtube.com/watch?v=7gEtKFweVBc) (12 min) — Traversy Media shows practical Docker development workflows
- [Docker for Development - Full Course](https://www.youtube.com/watch?v=0H2miBK_gAk) (20 min) — Practical guide to using Docker in development

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have created a Docker-based development setup for a simple project
- [ ] Changes to your local files are reflected inside the container in real time
- [ ] You understand how volumes make this possible
- [ ] You can explain when to use Docker for development vs. installing tools locally

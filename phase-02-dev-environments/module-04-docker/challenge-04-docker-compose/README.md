# Challenge: Docker Compose

## What you'll learn
How to use Docker Compose to define and run multi-container applications with a single command.

## Watch first
- [Docker Compose Tutorial](https://www.youtube.com/watch?v=SXwC9fSwct8) (13 min) — TechWorld with Nana explains Docker Compose clearly for beginners
- [Docker Compose in 12 Minutes](https://www.youtube.com/watch?v=Qw9zlE3t8Ko) (12 min) — Jake Wright gives a concise, practical intro to Docker Compose

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have written a `docker-compose.yml` file with at least 2 services
- [ ] You have started everything with `docker compose up`
- [ ] You have stopped everything with `docker compose down`
- [ ] You understand why Compose is better than running multiple `docker run` commands

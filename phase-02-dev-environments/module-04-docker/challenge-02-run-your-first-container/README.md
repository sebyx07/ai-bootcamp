# Challenge: Run Your First Container

## What you'll learn
How to pull Docker images, run containers, and interact with them using essential Docker commands.

## Watch first
- [Docker Tutorial for Beginners](https://www.youtube.com/watch?v=3c-iBn73dDE) (2hr 46min, watch 0:00-20:00) — TechWorld with Nana's excellent Docker fundamentals, just watch the first 20 minutes
- [Docker Crash Course for Absolute Beginners](https://www.youtube.com/watch?v=pg19Z8LL06w) (1hr, watch 0:00-15:00) — TechWorld with Nana condensed version, watch first 15 minutes

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have pulled an image with `docker pull`
- [ ] You have run an interactive container with `docker run -it`
- [ ] You have listed running containers with `docker ps`
- [ ] You have stopped a container with `docker stop`
- [ ] You have removed a container with `docker rm`

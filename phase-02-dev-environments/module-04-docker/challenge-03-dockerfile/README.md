# Challenge: Dockerfile

## What you'll learn
How to write a Dockerfile to build your own custom Docker image, turning any application into a portable container.

## Watch first
- [Dockerfile Tutorial - Docker in Practice](https://www.youtube.com/watch?v=WmcdMiyqfZs) (10 min) — TechWorld with Nana explains Dockerfiles step by step
- [Learn Docker in 7 Easy Steps](https://www.youtube.com/watch?v=gAkwW2tuIqE) (11 min) — Fireship covers Dockerfiles as part of a complete Docker overview

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have written a Dockerfile from scratch
- [ ] You have built an image with `docker build`
- [ ] You have run a container from your custom image
- [ ] You understand what each line of your Dockerfile does

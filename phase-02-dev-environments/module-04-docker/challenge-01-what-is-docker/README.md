# Challenge: What is Docker?

## What you'll learn
What Docker is, why it exists, and the difference between containers and virtual machines.

## Watch first
- [Docker in 100 Seconds](https://www.youtube.com/watch?v=Gjnup-PuquQ) (2 min) — Fireship explains Docker faster than anyone
- [you need to learn Docker RIGHT NOW!!](https://www.youtube.com/watch?v=eGz9DS-aIeY) (22 min) — NetworkChuck makes Docker fun and accessible for beginners

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what Docker does in your own words
- [ ] You understand the difference between a container and a virtual machine
- [ ] Docker is installed and `docker --version` works
- [ ] You have run `docker run hello-world` successfully

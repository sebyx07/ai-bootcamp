# Challenge: Install Redis

## What you'll learn
How to install Redis, a lightning-fast in-memory data store used for caching, sessions, and real-time features.

## Watch first
- [Redis in 100 Seconds](https://www.youtube.com/watch?v=G1rOthIU-uo) (2 min) — Fireship explains Redis at top speed
- [Redis Crash Course](https://www.youtube.com/watch?v=jgpVdJB2sKQ) (17 min) — Web Dev Simplified covers Redis fundamentals clearly

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] Redis is installed and the service is running
- [ ] You can connect using `redis-cli`
- [ ] You have stored and retrieved a value using SET and GET
- [ ] You can explain when you would use Redis vs PostgreSQL

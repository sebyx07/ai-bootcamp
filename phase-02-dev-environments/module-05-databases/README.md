# Module 05: Databases

## What you will learn

How to install and run PostgreSQL (a relational database) and Redis (an in-memory data store) on your machine.

## Why this matters

Almost every application needs to store data. PostgreSQL is the most popular open-source relational database, and Redis is used everywhere for caching and real-time features. Having both running locally means you can build real applications.

| Challenge | Topic |

| 02 | Install Redis -- a lightning-fast in-memory data store |

## Key concepts

- **PostgreSQL (Postgres)**: A relational database that stores data in tables with rows and columns
- **Redis**: An in-memory key-value store used for caching, sessions, and queues
- **Database server**: A program that runs in the background, waiting for connections
- **Port**: The network address a database listens on (Postgres uses 5432, Redis uses 6379)
- **Client**: A tool you use to connect to and interact with a database

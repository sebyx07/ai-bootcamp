# Challenge: Verify Everything Works

## What you'll learn
How to check that your entire development environment is properly set up and all tools are accessible.

## Watch first
- [How to Set Up a Development Environment](https://www.youtube.com/watch?v=4a2oFzGP71Y) (15 min) — Overview of what a complete dev environment looks like

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] All tools from this phase return version numbers when checked
- [ ] PostgreSQL and Redis services are running
- [ ] You have created a script that checks all tools at once
- [ ] You have a clear mental picture of everything installed on your machine

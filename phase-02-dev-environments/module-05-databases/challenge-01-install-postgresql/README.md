# Challenge: Install PostgreSQL

## What you'll learn
How to install PostgreSQL, the world's most advanced open-source relational database, and connect to it from the command line.

## Watch first
- [PostgreSQL in 100 Seconds](https://www.youtube.com/watch?v=n2Fluyr3lbc) (2 min) — Fireship explains Postgres incredibly fast
- [Learn PostgreSQL Tutorial - Full Course for Beginners](https://www.youtube.com/watch?v=qw--VYLpxG4) (4hr 19min, watch 0:00-15:00) — freeCodeCamp PostgreSQL course, just watch the first 15 minutes for setup

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] PostgreSQL is installed and the service is running
- [ ] You can connect to it using `psql`
- [ ] You have created a test database
- [ ] You have created a table and inserted a row of data

# Challenge: Install Something Useful

## What you'll learn
How to search for, evaluate, and install packages that solve real problems, building confidence with the package manager.

## Watch first
- [15 Useful Linux Commands Every Linux User Needs](https://www.youtube.com/watch?v=CVS_hOJrXcI) (12 min) — NetworkChuck shows useful command-line tools you can install

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have searched for packages using `apt search`
- [ ] You have installed at least 3 useful packages from a provided list
- [ ] You have tested each installed package to verify it works
- [ ] You can explain what each package does

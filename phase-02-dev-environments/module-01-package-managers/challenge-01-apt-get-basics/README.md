# Challenge: apt-get Basics

## What you'll learn
How to use apt, the default package manager on Ubuntu/Debian Linux, to update, install, and remove software from the command line.

## Watch first
- [Linux for Hackers - Package Management](https://www.youtube.com/watch?v=vX3krP6JmOY) (12 min) — NetworkChuck walks through apt and package management on Linux
- [Linux Crash Course - The apt Command](https://www.youtube.com/watch?v=1kicKTbK768) (10 min) — LearnLinuxTV explains apt commands clearly for beginners

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have run `sudo apt update` and understand what it does
- [ ] You have installed a package using `sudo apt install`
- [ ] You have removed a package using `sudo apt remove`
- [ ] You can explain the difference between `apt update` and `apt upgrade`

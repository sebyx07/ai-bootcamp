# Module 01: Package Managers

## What you will learn

How software gets installed on Linux. Instead of downloading programs from websites, Linux uses package managers -- tools that download, install, update, and remove software with a single command.

## Why this matters

Every tool you install in this entire bootcamp will come through a package manager. Understanding how they work means you will never be confused about where software comes from or how to update it.

## Key concepts

- **Package**: A bundle of software files packaged together for easy installation
- **Repository**: A server that stores thousands of packages you can install
- **Dependencies**: Other packages that a package needs in order to work
- **apt/apt-get**: The package manager used on Ubuntu and Debian-based Linux systems

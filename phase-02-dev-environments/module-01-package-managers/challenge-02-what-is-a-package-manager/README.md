# Challenge: What is a Package Manager?

## What you'll learn
The concept behind package managers -- why they exist, how they solve the dependency problem, and why every operating system has one.

## Watch first
- [How Do Linux Package Managers Work?](https://www.youtube.com/watch?v=UMJExMr3FBM) (8 min) — DistroTube explains how Linux package managers work under the hood
- [Package Managers - what are they and how do they work?](https://www.youtube.com/watch?v=PcEcKzBFxWk) (7 min) — Gary Explains breaks down the concept clearly

- [ ] You can explain what a package manager does in your own words
- [ ] You can name at least 3 different package managers and what systems use them

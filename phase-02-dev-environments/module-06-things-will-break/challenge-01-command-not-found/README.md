# Challenge: Command Not Found

## What you'll learn
How to diagnose and fix the dreaded "command not found" error, the single most common error beginners encounter.

## Watch first
- [Linux Command Not Found Error - How to Fix](https://www.youtube.com/watch?v=1Wjfos0Ywuk) (8 min) — Practical walkthrough of why "command not found" happens and how to fix it

- [ ] You have diagnosed a "command not found" error
- [ ] You understand at least 3 different reasons this error can happen
- [ ] You have fixed the broken scenario

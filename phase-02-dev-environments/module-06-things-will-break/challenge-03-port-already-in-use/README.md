# Challenge: Port Already in Use

## What you'll learn
How to diagnose and fix "port already in use" errors, which happen when two programs try to use the same network port.

## Watch first
- [What is a Port?](https://www.youtube.com/watch?v=g2fT-g9PX9o) (5 min) — Quick explanation of what network ports are and why they matter
- [How to Fix "Port Already in Use" Error](https://www.youtube.com/watch?v=AXoaOmGMb3o) (6 min) — Practical guide to finding and killing processes on ports

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have encountered a "port already in use" error
- [ ] You can find which process is using a specific port
- [ ] You have freed up the port by stopping the process
- [ ] You understand what ports are and why two programs cannot share one

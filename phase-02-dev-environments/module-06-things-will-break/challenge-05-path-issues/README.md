# Challenge: PATH Issues

## What you'll learn
How the PATH environment variable works, why it causes so many problems, and how to fix it when things go wrong.

## Watch first
- [Understanding the Linux PATH Variable](https://www.youtube.com/watch?v=jIHnGQrqsmo) (10 min) — Clear explanation of what PATH is and how it works
- [Environment Variables Explained](https://www.youtube.com/watch?v=ADh_OFBfdEE) (7 min) — Fireship explains environment variables including PATH

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You can explain what the PATH variable is and what it does
- [ ] You have diagnosed a PATH-related problem
- [ ] You know how to add a directory to your PATH (temporarily and permanently)
- [ ] You understand the order of directories in PATH and why it matters

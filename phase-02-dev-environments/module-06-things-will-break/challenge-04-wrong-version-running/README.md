# Challenge: Wrong Version Running

## What you'll learn
How to diagnose and fix issues where the wrong version of a tool is active, causing mysterious failures in your projects.

## Watch first
- [The Dangers of Different Node Versions](https://www.youtube.com/watch?v=JQSflHiFJEU) (7 min) — Real-world example of version mismatch problems
- [How to Use NVM to Manage Node.js Versions](https://www.youtube.com/watch?v=ohBFbA0O6hs) (8 min) — Review of version switching with NVM

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have diagnosed a version mismatch problem
- [ ] You know how to check which version of a tool is actually running
- [ ] You understand how `which` and version managers interact
- [ ] You have fixed the version and verified the correct one is active

# Challenge: Permission Denied

## What you'll learn
How to diagnose and fix "permission denied" errors, which happen when you try to do something your user account is not allowed to do.

## Watch first
- [Linux File Permissions in 5 Minutes](https://www.youtube.com/watch?v=LnKoncbQBsM) (5 min) — Quick clear explanation of Linux permissions
- [Linux Permissions - Everything You Need to Know](https://www.youtube.com/watch?v=4e669hSjaX8) (17 min) — More detailed dive into how permissions work

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have encountered and fixed a "permission denied" error
- [ ] You can read basic file permissions using `ls -l`
- [ ] You understand when to use `sudo` and when NOT to
- [ ] You know the difference between `chmod` and `chown`

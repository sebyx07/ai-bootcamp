# Challenge: Install Ruby

## What you'll learn
How to use rbenv to install a specific version of Ruby and verify it is working.

## Watch first
- [Ruby Programming Language - Full Course](https://www.youtube.com/watch?v=t_ispmWmdjY) (4 hr, watch first 10 min) — freeCodeCamp intro to Ruby, just watch the setup portion
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (2 min) — Quick refresher on what Ruby is

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have installed Ruby using rbenv
- [ ] Running `ruby --version` shows the version you installed
- [ ] You have run `ruby -e 'puts "Hello from Ruby!"'` successfully
- [ ] Running `which ruby` shows a path inside your `.rbenv` directory

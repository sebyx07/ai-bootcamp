# Challenge: Install rbenv

## What you'll learn
How to install rbenv, a lightweight Ruby version manager that works similarly to NVM but for the Ruby programming language.

## Watch first
- [How to Install Ruby on Linux with rbenv](https://www.youtube.com/watch?v=NvMaAD1euSk) (10 min) — Step-by-step rbenv installation on Linux
- [Ruby in 100 Seconds](https://www.youtube.com/watch?v=UYm0kfnRTJk) (2 min) — Fireship gives a lightning-fast overview of Ruby

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] rbenv is installed on your system
- [ ] Running `rbenv --version` shows a version number
- [ ] You understand the pattern: version managers let you control which language version is active

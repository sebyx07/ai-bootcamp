# Module 03: Ruby and rbenv

## What you will learn

How to install Ruby using rbenv (a Ruby version manager), and how Ruby's package ecosystem (gems and Bundler) works.

## Why this matters

Ruby is the language behind Ruby on Rails, one of the most popular web frameworks. Even if you work with other languages, understanding how Ruby manages versions and packages teaches you patterns you will see everywhere.

## Challenges

| Challenge | Topic |
|-----------|-------|
| 01 | Install rbenv -- the tool that manages Ruby versions |
| 02 | Install Ruby -- get the language running |
| 03 | Gems and Bundler -- Ruby's package management system |

## Key concepts

- **Ruby**: A programming language designed for developer happiness
- **rbenv**: Ruby Environment -- installs and switches between Ruby versions
- **Gems**: Ruby's name for packages/libraries
- **Bundler**: A tool that manages which gems a project needs (like npm for Ruby)
- **Gemfile**: A file that lists all the gems a project depends on

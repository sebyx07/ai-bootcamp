# Challenge: Gems and Bundler

## What you'll learn
How Ruby's package ecosystem works -- gems are Ruby's packages, and Bundler manages which gems a project needs.

## Watch first
- [What are Ruby Gems?](https://www.youtube.com/watch?v=IEkPSKSbHEc) (6 min) — Clear beginner explanation of Ruby gems
- [Bundler in 5 Minutes](https://www.youtube.com/watch?v=4DqzaqeeMgY) (5 min) — Quick intro to what Bundler does and why you need it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have installed a gem using `gem install`
- [ ] You understand what a Gemfile is
- [ ] You have created a project directory with a Gemfile and run `bundle install`
- [ ] You can explain how gems relate to npm packages

# Challenge: Install NVM

## What you'll learn
How to install NVM (Node Version Manager), the tool that lets you install and manage multiple versions of Node.js.

## Watch first
- [NVM - Install and Use Node Version Manager](https://www.youtube.com/watch?v=ohBFbA0O6hs) (8 min) — Quick walkthrough of installing and using NVM on Linux
- [Node.js Ultimate Beginner's Guide in 7 Easy Steps](https://www.youtube.com/watch?v=ENrzD9HAZK4) (11 min) — Fireship gives a fast overview of what Node.js is and why it matters

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] NVM is installed on your system
- [ ] Running `nvm --version` shows a version number
- [ ] You understand why we use NVM instead of installing Node directly

# Challenge: Install Node

## What you'll learn
How to use NVM to install Node.js and verify that both Node and npm (Node's package manager) are working.

## Watch first
- [Node.js in 100 Seconds](https://www.youtube.com/watch?v=ENrzD9HAZK4) (2 min) — Fireship explains Node.js incredibly quickly
- [What is npm? Node Package Manager](https://www.youtube.com/watch?v=ZNbFagCBlwo) (8 min) — Quick explanation of npm and why you need it

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have installed Node.js LTS using NVM
- [ ] Running `node --version` shows a version number
- [ ] Running `npm --version` shows a version number
- [ ] You have run a simple JavaScript command with `node -e "console.log('Hello from Node!')"`

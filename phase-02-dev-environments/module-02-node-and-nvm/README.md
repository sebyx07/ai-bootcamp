# Module 02: Node and NVM

## What you will learn

How to install Node.js (the JavaScript runtime) using NVM (Node Version Manager), and why version management matters.

## Why this matters

Node.js is everywhere. Even if you do not plan to write JavaScript, many development tools (build tools, linters, package managers like npm) depend on it. Different projects need different versions of Node, so you need a way to switch between them.

## Challenges

| Challenge | Topic |
|-----------|-------|
| 01 | Install NVM -- the tool that manages Node versions |
| 02 | Install Node -- get your first JavaScript runtime |
| 03 | Switch versions -- learn why version management exists |

## Key concepts

- **Node.js**: A runtime that lets you run JavaScript outside a web browser
- **NVM**: Node Version Manager -- installs and switches between Node versions
- **npm**: Node Package Manager -- installs JavaScript libraries (comes with Node)
- **LTS**: Long Term Support -- the stable, recommended version of Node

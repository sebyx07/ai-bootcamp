# Challenge: Switch Node Versions

## What you'll learn
How to install multiple versions of Node.js and switch between them, which is essential when working on different projects.

## Watch first
- [How to Use NVM to Manage Node.js Versions](https://www.youtube.com/watch?v=ohBFbA0O6hs) (8 min) — Demonstrates switching between Node versions with NVM

## Open Claude Code and say:
> "I'm ready for the challenge"

## You're done when
- [ ] You have at least 2 different versions of Node installed
- [ ] You can switch between them using `nvm use`
- [ ] You have set a default version with `nvm alias default`
- [ ] You can list all installed versions with `nvm ls`
- [ ] You understand why a project might need a specific Node version
